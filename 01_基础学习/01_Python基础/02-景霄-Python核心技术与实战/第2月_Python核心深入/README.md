## 💡 学习建议

### 学习方法
1. **装饰器**：先理解"函数是第一类对象"，再看装饰器本质是"高阶函数返回函数"
2. **并发**：从GIL限制入手，理解为何Python多线程不真正并行、何时用多进程/协程
3. **数据结构**：手工画图理解链表指针、栈的LIFO、队列的FIFO原理
4. **LeetCode**：每做完一题，总结解题模式（双指针、哈希表、滑动窗口等）

### 时间分配
- 每日2-3小时，周末可增至4-5小时
- 周一~周四：理论学习 + 编码练习
- 周五：复习本周内容 + 查漏补缺
- 周六：LeetCode集中刷题
- 周日：项目实践 + 知识整理

### 心态调整
- 第2月是**Python深度提升期**，可能会遇到"知识点多、难以消化"的阶段
- 装饰器和并发是面试高频考点，务必反复练习
- LeetCode初期不要追求速度，重在理解算法思想
- 遇到卡点时，回到基础概念，画图辅助理解
