### 5. 复杂装饰器：缓存 + 过期时间

- 缓存键的生成（基于函数名和参数）
- 过期时间的处理
- 线程安全（可选）

要求：
```python

def expensive_computation(x):
    import time
    time.sleep(1)  # 模拟耗时计算
    return x * 2

# 第一次调用耗时1秒
print(expensive_computation(10))  # 耗时1秒，输出20

# 5秒内再次调用立即返回缓存
print(expensive_computation(10))  # 立即返回20

# 5秒后调用重新计算
import time
time.sleep(6)
print(expensive_computation(10))  # 再次耗时1秒
```
