## 第一部分：选择题答案解析

### 1. 数组与链表区别
**正确答案：A**  
数组在内存中连续存储，支持O(1)随机访问；链表通过指针连接，不一定连续。链表随机访问需要O(n)（B错），数组插入/删除平均O(n)（C错），链表有额外指针开销（D错）。

### 2. 栈的特性
**正确答案：B**  
栈遵循LIFO（后进先出）原则，像叠盘子。队列是FIFO（先进先出）。

### 3. 队列的应用场景
**正确答案：C**  
打印任务排队是典型的队列应用（先进先出）。函数调用栈用栈（A），浏览器前进后退用两个栈（B），表达式求值用栈（D）。

### 4. 哈希表冲突解决
**正确答案：D**  
二叉树法不是哈希表解决冲突的标准方法。标准方法：链地址法、开放地址法（线性探测、二次探测、双重哈希）、再哈希法。

### 5. 二叉树遍历
**正确答案：A**  
前序遍历：根-左-右；中序遍历：左-根-右；后序遍历：左-右-根；层序遍历：按层次从上到下。

### 6. BFS vs DFS
**正确答案：C**  
BFS使用队列（A错），DFS使用栈/递归（B错）。BFS适合无权图最短路径，DFS可能更省内存（递归深度）（D不完全正确）。

### 7. LeetCode Easy题特征
**正确答案：C**  
Easy题通常是数组/字符串基本操作、简单数学、基础数据结构。动态规划（A）、图论（B）、红黑树（D）一般是Medium/Hard。

### 8. 两数之和（Two Sum）
**正确答案：D**  
三种解法都可行，但最优是哈希表O(n)。暴力枚举O(n²)最简单，双指针需要先排序O(n log n)。

### 9. 反转链表（Reverse Linked List）
**正确答案：C**  
需要遍历每个节点一次，时间复杂度O(n)。空间复杂度可以是O(1)（迭代）或O(n)（递归栈）。

### 10. 有效括号（Valid Parentheses）
**正确答案：C**  
栈最适合处理括号匹配：遇到左括号入栈，遇到右括号检查栈顶是否匹配。

### 11. 链表环检测（Linked List Cycle）
**正确答案：A**  
Floyd判圈算法（快慢指针）时间复杂度O(n)，空间复杂度O(1)。哈希表法也是O(n)但需要O(n)空间。

### 12. 合并两个有序数组（Merge Sorted Array）
**正确答案：A**  
归并排序的合并步骤，需要遍历两个数组各一次，O(m+n)。

### 13. 最大子数组和（Maximum Subarray）
**正确答案：A**  
Kadane算法动态规划，一次遍历O(n)。分治法O(n log n)不是最优。

### 14. 二叉树的最大深度（Maximum Depth of Binary Tree）
**正确答案：C**  
BFS和DFS都可以，都需要访问每个节点一次，时间复杂度O(n)。BFS需要队列存储节点，空间复杂度可能更高。

### 15. 对称二叉树（Symmetric Tree）
**正确答案：A**  
对称二叉树要求左右子树镜像对称。常用递归：比较左子树的左孩子和右子树的右孩子，左子树的右孩子和右子树的左孩子。

### 1. 实现基本数据结构

**a) 链表实现**
```python
class ListNode:
    def __init__(self, val=0):
        self.val = val
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None
    
    def append(self, val):
        """在链表末尾添加节点"""
        new_node = ListNode(val)
        if not self.head:
            self.head = new_node
            return
        
        current = self.head
        while current.next:
            current = current.next
        current.next = new_node
    
    def prepend(self, val):
        """在链表头部添加节点"""
        new_node = ListNode(val)
        new_node.next = self.head
        self.head = new_node
    
    def delete(self, val):
        """删除第一个值为val的节点"""
        if not self.head:
            return
        
        # 如果要删除的是头节点
        if self.head.val == val:
            self.head = self.head.next
            return
        
        # 查找要删除的节点
        current = self.head
        while current.next and current.next.val != val:
            current = current.next
        
        # 如果找到，删除节点
        if current.next:
            current.next = current.next.next
    
    def display(self):
        """打印链表"""
        values = []
        current = self.head
        while current:
            values.append(current.val)
            current = current.next
        print(" -> ".join(map(str, values)) if values else "Empty list")
```

**b) 栈实现（使用链表）**
```python
class StackNode:
    def __init__(self, val):
        self.val = val
        self.next = None

class Stack:
    def __init__(self):
        self.top = None
    
    def push(self, val):
        """入栈"""
        new_node = StackNode(val)
        new_node.next = self.top
        self.top = new_node
    
    def pop(self):
        """出栈"""
        if not self.top:
            raise IndexError("Pop from empty stack")
        val = self.top.val
        self.top = self.top.next
        return val
    
    def peek(self):
        """查看栈顶元素"""
        if not self.top:
            raise IndexError("Peek from empty stack")
        return self.top.val
    
    def is_empty(self):
        """判断栈是否为空"""
        return self.top is None
    
    def display(self):
        """打印栈（从上到下）"""
        values = []
        current = self.top
        while current:
            values.append(current.val)
            current = current.next
        print("Stack (top to bottom):", values if values else "Empty")
```

### 2. LeetCode 1：两数之和（Two Sum）
```python
def two_sum(nums, target):
    """
    哈希表解法：时间复杂度O(n)，空间复杂度O(n)
    """
    num_map = {}  # 值 -> 索引
    
    for i, num in enumerate(nums):
        complement = target - num
        if complement in num_map:
            return [num_map[complement], i]
        num_map[num] = i
    
    return []  # 根据题目假设，这里不会执行

# 测试
print(two_sum([2, 7, 11, 15], 9))  # [0, 1]
print(two_sum([3, 2, 4], 6))       # [1, 2]
print(two_sum([3, 3], 6))          # [0, 1]

# 暴力枚举解法（对比）
def two_sum_brute_force(nums, target):
    """暴力枚举：时间复杂度O(n²)，空间复杂度O(1)"""
    n = len(nums)
    for i in range(n):
        for j in range(i + 1, n):
            if nums[i] + nums[j] == target:
                return [i, j]
    return []
```

### 3. LeetCode 20：有效的括号（Valid Parentheses）
```python
def is_valid(s):
    """
    栈解法：时间复杂度O(n)，空间复杂度O(n)
    """
    stack = []
    mapping = {')': '(', '}': '{', ']': '['}
    
    for char in s:
        if char in mapping:  # 右括号
            top_element = stack.pop() if stack else '#'
            if mapping[char] != top_element:
                return False
        else:  # 左括号
            stack.append(char)
    
    return not stack  # 栈空则有效

# 测试
test_cases = ["()", "()[]{}", "(]", "([)]", "{[]}", "", "((()))"]
for test in test_cases:
    print(f"{test:10}: {is_valid(test)}")

# 更健壮的版本
def is_valid_robust(s):
    if not s:
        return True
    
    stack = []
    for char in s:
        if char in '({[':
            stack.append(char)
        elif char in ')}]':
            if not stack:
                return False
            top = stack.pop()
            if (char == ')' and top != '(') or \
               (char == '}' and top != '{') or \
               (char == ']' and top != '['):
                return False
        else:
            # 非法字符
            return False
    
    return len(stack) == 0
```

### 4. LeetCode 21：合并两个有序链表（Merge Two Sorted Lists）
```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def merge_two_lists(l1, l2):
    """
    迭代解法：时间复杂度O(m+n)，空间复杂度O(1)
    """
    dummy = ListNode(0)  # 虚拟头节点
    current = dummy
    
    while l1 and l2:
        if l1.val <= l2.val:
            current.next = l1
            l1 = l1.next
        else:
            current.next = l2
            l2 = l2.next
        current = current.next
    
    # 连接剩余部分
    current.next = l1 if l1 else l2
    
    return dummy.next

# 递归解法
def merge_two_lists_recursive(l1, l2):
    """递归解法：时间复杂度O(m+n)，空间复杂度O(m+n)（递归栈）"""
    if not l1:
        return l2
    if not l2:
        return l1
    
    if l1.val <= l2.val:
        l1.next = merge_two_lists_recursive(l1.next, l2)
        return l1
    else:
        l2.next = merge_two_lists_recursive(l1, l2.next)
        return l2

# 辅助函数
def create_list(arr):
    if not arr:
        return None
    head = ListNode(arr[0])
    current = head
    for val in arr[1:]:
        current.next = ListNode(val)
        current = current.next
    return head

def print_list(head):
    values = []
    while head:
        values.append(head.val)
        head = head.next
    print(values)

# 测试
l1 = create_list([1, 2, 4])
l2 = create_list([1, 3, 4])
merged = merge_two_lists(l1, l2)
print_list(merged)  # [1, 1, 2, 3, 4, 4]
```

### 5. LeetCode 104：二叉树的最大深度（Maximum Depth of Binary Tree）
```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def max_depth(root):
    """
    递归解法：时间复杂度O(n)，空间复杂度O(h)（递归栈深度）
    """
    if not root:
        return 0
    
    left_depth = max_depth(root.left)
    right_depth = max_depth(root.right)
    
    return max(left_depth, right_depth) + 1

# BFS解法（层序遍历）
def max_depth_bfs(root):
    """BFS解法：时间复杂度O(n)，空间复杂度O(w)（最大宽度）"""
    if not root:
        return 0
    
    from collections import deque
    queue = deque([root])
    depth = 0
    
    while queue:
        depth += 1
        level_size = len(queue)
        for _ in range(level_size):
            node = queue.popleft()
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
    
    return depth

# 测试二叉树
def create_test_tree():
    root = TreeNode(3)
    root.left = TreeNode(9)
    root.right = TreeNode(20)
    root.right.left = TreeNode(15)
    root.right.right = TreeNode(7)
    return root

# 测试
tree = create_test_tree()
print("递归解法:", max_depth(tree))          # 3
print("BFS解法:", max_depth_bfs(tree))      # 3
print("空树:", max_depth(None))             # 0
```

## 第三部分：数据结构概念题答案

### 1. 时间复杂度分析
- **数组随机访问**：O(1) - 通过索引直接计算内存地址
- **链表查找元素**：O(n) - 需要从头遍历
- **哈希表查找元素**：O(1)平均情况，O(n)最坏情况（哈希冲突）
- **栈的入栈操作**：O(1) - 在栈顶添加元素
- **队列的出队操作**：O(1) - 从队头移除元素（如果使用链表实现）

### 2. 空间复杂度比较
- **数组 vs 链表**：数组需要连续内存，可能浪费空间；链表每个节点有额外指针开销
- **哈希表 vs 二叉树**：哈希表有装载因子，可能浪费部分槽位；二叉树每个节点有两个指针
- **栈（数组实现）vs 栈（链表实现）**：数组栈可能预分配过多空间；链表栈每个元素有指针开销

### 3. 实际应用场景匹配
1. **浏览器前进后退**：A. 栈（两个栈实现）
2. **操作系统进程调度**：B. 队列（如就绪队列）
3. **电话通讯录快速查找**：C. 哈希表（按姓名快速查找）
4. **表达式求值**：A. 栈（运算符栈和操作数栈）
5. **社交网络好友关系**：D. 图（节点表示人，边表示关系）

### 4. 算法策略选择
1. **查找数组中的最大值**：A. 线性扫描（O(n)遍历）
2. **判断字符串是否回文**：B. 双指针（从两端向中间比较）
3. **查找排序数组中的元素**：C. 二分查找（O(log n)）
4. **计算斐波那契数列**：D. 动态规划（避免重复计算）
5. **排序整数数组**：E. 快速排序（平均O(n log n)）

## 🔍 核心知识点总结

### 数据结构基础要点
1. **数组**：连续内存，随机访问O(1)，插入删除O(n)
2. **链表**：非连续，通过指针连接，插入删除O(1)，查找O(n)
3. **栈**：LIFO，适合括号匹配、函数调用、DFS
4. **队列**：FIFO，适合BFS、任务调度、缓冲
5. **哈希表**：平均O(1)查找，解决冲突（链地址法、开放地址法）
6. **树**：层次结构，二叉树遍历（前中后序、层序）

### LeetCode刷题要点
1. **Easy题目标**：理解基础算法，熟练编码，注意边界条件
2. **解题步骤**：理解题意 → 设计算法 → 编写代码 → 测试调试
3. **时间/空间复杂度**：分析算法效率，追求最优解
4. **常见模式**：双指针、哈希表、栈、递归、BFS/DFS

### 算法思维培养
1. **问题分解**：将复杂问题分解为子问题
2. **模式识别**：识别问题属于哪类（搜索、排序、动态规划等）
3. **优化意识**：从暴力解法开始，逐步优化
4. **测试驱动**：编写测试用例，验证算法正确性

### 常见陷阱与解决方案
1. **空指针/空数组**：始终检查边界条件
2. **索引越界**：注意循环条件（`< len` vs `<= len`）
3. **递归深度**：Python默认递归深度限制（约1000）
4. **大数运算**：注意整数溢出（Python自动处理大整数）
5. **时间复杂度**：避免嵌套循环导致O(n²)
