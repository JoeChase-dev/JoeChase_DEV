## 5.1 函数是"一等公民"

Python 中函数具有以下能力：

| 能力 | 说明 | 示例 |
|:-----|:-----|:-----|
| 赋值给变量 | 函数可以当值用 | `f = len` |
| 作为参数传入 | 高阶函数基础 | `map(f, lst)` |
| 作为返回值 | 闭包的基础 | `def outer(): return inner` |

---

## 5.2 高阶函数

### map（映射）

```python
# map(函数, 可迭代对象) → 将每个元素依次传入函数，返回迭代器
map(lambda x: x * x, [1, 2, 3, 4, 5])
# 等价于：[1, 4, 9, 16, 25]
```

### filter（过滤）

```python
# filter(函数, 可迭代对象) → 只保留使函数返回 True 的元素
filter(lambda x: x % 2, range(1, 11))
# 等价于：[1, 3, 5, 7, 9]  （只保留奇数）
```

**⚠️ 注意**：map 和 filter 返回的是**迭代器**（惰性计算），需要 `list()` 转为列表。

```python
# 组合使用
items = list(map(lambda x: x ** 2, filter(lambda x: x % 2, range(1, 11))))
# [1, 9, 25, 49, 81]
```

## 5.3 Lambda 匿名函数

```python
# lambda 参数列表: 表达式
lambda x: x ** 2          # 一个参数
lambda x, y: x + y        # 多个参数
lambda x: x['price']      # 用于排序 key
```

**特点**：
- 匿名（不需要 def）
- 单表达式（不能写多行语句）
- 常用于高阶函数的短回调

## 5.4 参数类型总结

| 参数类型 | 语法 | 说明 |
|:--------|:-----|:-----|
| 位置参数 | `def f(a, b)` | 按位置传递 |
| 默认参数 | `def f(a=10)` | 有默认值 |
| 可变位置参数 | `def f(*args)` | 收集多余位置参数为元组 |
| 可变关键字参数 | `def f(**kwargs)` | 收集多余关键字参数为字典 |
| 命名关键字参数 | `def f(*, a, b)` | 必须用 `关键字` 传参 |
| 仅位置参数 | `def f(a, /, b)` | a 只能按位置传（Python 3.8+） |

## 5.5 参数解包

```python
def input_thing():
    name_str, price_str, weight_str = input().split()
    return name_str, int(price_str), int(weight_str)   # 返回元组

thing = Thing(*input_thing())
# * 操作符将元组"解包"为独立参数
# ('电脑', 200, 20) → Thing('电脑', 200, 20)
```
