## 2.1 heapq 模块 — 堆/优先队列

```python
import heapq

list1 = [34, 25, 18, 87, 76, 95, 48, 65, 28, 14]

print(heapq.nlargest(3, list1))    # [95, 87, 76]   ← 最大的3个
print(heapq.nsmallest(3, list1))   # [14, 18, 25]   ← 最小的3个
```

### 配合 lambda 使用

```python
list2 = [
    {'name': 'IBM',     'shares': 100, 'price': 91.1},
    {'name': 'AAPL',    'shares': 50,  'price': 543.22},
    {'name': 'FB',      'shares': 200, 'price': 21.09},
    {'name': 'HPQ',     'shares': 35,  'price': 31.75},
    {'name': 'YHOO',    'shares': 45,  'price': 16.35},
    {'name': 'ACME',    'shares': 75,  'price': 115.65}
]

# key 参数可以是函数、方法或 lambda 表达式
print(heapq.nlargest(2, list2, key=lambda x: x['price']))
# [{'name': 'AAPL', ...}, {'name': 'ACME', ...}]

print(heapq.nsmallest(2, list2, key=lambda x: x['price']))
# [{'name': 'YHOO', ...}, {'name': 'FB', ...}]
```

**应用场景**：从列表中找出最大或最小的 N 个元素（堆结构，O(n log k) 效率）。

## 2.2 itertools 模块 — 迭代器工具库

### permutations（全排列）— 有序

```python
import itertools

for perm in itertools.permutations('ABCD', 2):
    print(perm)
# ('A', 'B') ('A', 'C') ('A', 'D')
# ('B', 'A') ('B', 'C') ('B', 'D')
# ('C', 'A') ('C', 'B') ('C', 'D')
# ('D', 'A') ('D', 'B') ('D', 'C')
```

**特点**：AB 和 BA 算两个不同的排列（有序）

### combinations（组合）— 无序

```python
for comb in itertools.combinations('ABCDE', 2):
    print(comb)
# ('A', 'B') ('A', 'C') ('A', 'D') ('A', 'E')
# ('B', 'C') ('B', 'D') ('B', 'E')
# ('C', 'D') ('C', 'E')
# ('D', 'E')
```

**特点**：AB 和 BA 算同一个组合（无序）

### product（笛卡尔积）— 所有组合

```python
for prod in itertools.product('ABCD', '123'):
    print(prod)
# ('A', '1') ('A', '2') ('A', '3')
# ('B', '1') ...
# 共 12 种组合
```

**等价于嵌套循环**：
```python
for x in 'ABCD':
    for y in '123':
        print((x, y))
```

### cycle（无限循环迭代器）

```python
counter = 0
for item in itertools.cycle(('A', 'B', 'C')):
    print(item, end=' ')
    counter += 1
    if counter >= 10:
        break
# A B C A B C A B C A 
```

## 2.3 collections 模块 — 高级容器类型

| 类 | 说明 | 特点 |
|:---|:-----|:-----|
| `namedtuple` | 命名元组 | 类工厂，创建带属性名的元组 |
| `deque` | 双端队列 | 底层双向链表，头尾操作 O(1) |
| **`Counter`** | 计数器 | dict 子类，统计元素频率 |
| `OrderedDict` | 有序字典 | 记录键值对插入顺序 |
| `defaultdict` | 带默认值的字典 | 比 setdefault() 更高效 |

### Counter 示例

```python
from collections import Counter

word = [
    'look', 'into', 'my', 'eyes', 'look', 'into', 'my', 'eyes',
    'the', 'eyes', 'the', 'eyes', 'the', 'eyes', 'not', 'around',
    'the', 'eyes', "don't", 'look', 'around', 'the', 'eyes',
    'look', 'into', 'my', 'eyes', "you're", 'under'
]

counter = Counter(word)
print(counter.most_common(3))
# [('eyes', 8), ('the', 5), ('look', 4)]
```
