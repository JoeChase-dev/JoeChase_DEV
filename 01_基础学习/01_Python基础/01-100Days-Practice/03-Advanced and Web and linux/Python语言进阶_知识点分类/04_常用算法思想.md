## 算法思想总览

| 算法 | 思想 | 特点 |
|:-----|:-----|:-----|
| **穷举法** | 暴力破解，验证所有可能性 | 最直观，但效率低 |
| **贪婪法** | 每步选当前最优，不追求全局最优 | 快速找到满意解 |
| **分治法** | 分解→解决→合并 | 递归思想 |
| **回溯法** | 向前试探，不行就退回重选 | 搜索+剪枝 |
| **动态规划** | 保存子问题结果，避免重复计算 | 以空间换时间 |

## 4.1 穷举法（暴力破解）

### 示例1：百钱百鸡

```python
# 公鸡5元一只，母鸡3元一只，小鸡1元三只
# 100元买100只鸡，各多少只？
for x in range(20):          # 公鸡最多20只
    for y in range(33):      # 母鸡最多33只
        z = 100 - x - y
        if 5 * x + 3 * y + z // 3 == 100 and z % 3 == 0:
            print(x, y, z)
```

### 示例2：分鱼问题

```python
# 5人捕鱼，每次分5份扔掉1条拿走自己的
fish = 6
while True:
    total = fish
    enough = True
    for _ in range(5):
        if (total - 1) % 5 == 0:
            total = (total - 1) // 5 * 4
        else:
            enough = False
            break
    if enough:
        print(fish)     # 至少捕了 3121 条
        break
    fish += 5
```

## 4.2 贪婪法

### 示例：背包问题

```python
class Thing(object):
    """物品"""
    def __init__(self, name, price, weight):
        self.name = name
        self.price = price
        self.weight = weight

    @property
    def value(self):
        """价格重量比（性价比）"""
        return self.price / self.weight


def main():
    max_weight, num_of_things = map(int, input().split())
    all_things = []
    for _ in range(num_of_things):
        # * 解包操作符：元组 → 多参数
        all_things.append(Thing(*input_thing()))

    # 按性价比降序排列
    all_things.sort(key=lambda x: x.value, reverse=True)

    total_weight, total_price = 0, 0
    for thing in all_things:
        if total_weight + thing.weight <= max_weight:
            print(f'小偷拿走了{thing.name}')
            total_weight += thing.weight
            total_price += thing.price
    print(f'总价值: {total_price}美元')


if __name__ == '__main__':
    main()
```

## 4.3 分治法

### 示例：快速排序

```python
def quick_sort(items, comp=lambda x, y: x <= y):
    items = list(items)[:]
    _quick_sort(items, 0, len(items) - 1, comp)
    return items


def _quick_sort(items, start, end, comp):
    if start < end:
        pos = _partition(items, start, end, comp)
        _quick_sort(items, start, pos - 1, comp)   # 排左半部分
        _quick_sort(items, pos + 1, end, comp)     # 排右半部分


def _partition(items, start, end, comp):
    pivot = items[end]       # 选最后一个元素作为枢轴
    i = start - 1
    for j in range(start, end):
        if comp(items[j], pivot):
            i += 1
            items[i], items[j] = items[j], items[i]
    items[i + 1], items[end] = items[end], items[i + 1]
    return i + 1
```

**分治核心**：选择枢轴 → 划分（左边小/右边大） → 递归处理左右两段。

## 4.4 回溯法

### 示例：骑士巡逻（递归回溯）

```python
import sys
import time

SIZE = 5
total = 0

def print_board(board):
    for row in board:
        for col in row:
            print(str(col).center(4), end='')
        print()

def patrol(board, row, col, step=1):
    """递归回溯：8个方向试探"""
    if 0 <= row < SIZE and 0 <= col < SIZE and board[row][col] == 0:
        board[row][col] = step
        if step == SIZE * SIZE:          # 走完了！
            global total
            total += 1
            print(f'第{total}种走法:')
            print_board(board)
        # 8个方向递归试探（日字跳）
        patrol(board, row - 2, col - 1, step + 1)
        patrol(board, row - 1, col - 2, step + 1)
        patrol(board, row + 1, col - 2, step + 1)
        patrol(board, row + 2, col - 1, step + 1)
        patrol(board, row + 2, col + 1, step + 1)
        patrol(board, row + 1, col + 2, step + 1)
        patrol(board, row - 1, col + 2, step + 1)
        patrol(board, row - 2, col + 1, step + 1)
        board[row][col] = 0              # 回溯！撤销这步，尝试别的路
```

**关键**：`board[row][col] = 0` 是回溯的核心——走不通就退回来换条路。

## 4.5 动态规划

### 示例：子列表最大和

```python
def main():
    items = list(map(int, input().split()))
    overall = partial = items[0]
    for i in range(1, len(items)):
        partial = max(items[i], partial + items[i])   # 要么重新开始，要么接上
        overall = max(partial, overall)               # 记录历史最大值
    print(overall)


# 输入：1 -2 3 5 -3 2   → 输出：8
# 输入：-9 -2 -3 -5 -3  → 输出：-2
```

**核心思想**：
- `partial`：以当前位置结尾的最大子列和
- `overall`：全局最大子列和
- 时间复杂度从 O(n²) → O(n)，只需两个变量！

**状态转移方程**：`dp[i] = max(nums[i], dp[i-1] + nums[i])`
