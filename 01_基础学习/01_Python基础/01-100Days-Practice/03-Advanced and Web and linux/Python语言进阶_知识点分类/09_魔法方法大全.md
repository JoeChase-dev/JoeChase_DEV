## 9.1 什么是魔法方法？

以 `__` 开头和结尾的方法，也叫 **双下方法（dunder methods）**。

它们是 Python 的**钩子**——你不需要直接调用，Python 在特定场景下自动调用它们。

### 四大核心问题

| 问题 | 答案 | 需要实现 |
|:-----|:-----|:---------|
| 自定义对象能用运算符吗？ | ✅ 能 | `__add__`, `__eq__` 等 |
| 能放入 set 并去重吗？ | ✅ 能 | `__hash__` + `__eq__` |
| 能作为 dict 键吗？ | ✅ 能 | `__hash__` + `__eq__` |
| 能用 with 语法吗？ | ✅ 能 | `__enter__` + `__exit__` |

## 9.2 表示相关 — 控制对象如何被打印

| 方法 | 触发时机 | 用途 |
|:-----|:---------|:-----|
| `__str__` | `print(obj)` / `str(obj)` | 面向用户的友好字符串 |
| `__repr__` | `repr(obj)` / 交互式终端 | 面向开发者的精确描述 |

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score
    
    def __str__(self):
        return f'【学生】{self.name}，成绩：{self.score}'
    
    def __repr__(self):
        return f'Student(name={self.name!r}, score={self.score})'
```

## 9.3 运算符重载

| 方法 | 对应运算符 | 说明 |
|:-----|:----------|:-----|
| `__add__` | `a + b` | 加法 |
| `__sub__` | `a - b` | 减法 |
| `__mul__` | `a * b` | 乘法 |
| `__truediv__` | `a / b` | 除法 |
| `__eq__` | `a == b` | 等于 |
| `__lt__`, `__le__` | `<`, `<=` | 小于、小于等于 |
| `__gt__`, `__ge__` | `>`, `>=` | 大于、大于等于 |
| `__len__` | `len(obj)` | 长度 |
| `__getitem__` | `obj[key]` | 索引访问 |
| `__setitem__` | `obj[key] = val` | 索引赋值 |
| `__contains__` | `x in obj` | 成员判断 |

```python
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    
    def __add__(self, other):       # 实现 +
        return Vector(self.x + other.x, self.y + other.y)
    
    def __eq__(self, other):        # 实现 ==
        return self.x == other.x and self.y == other.y
    
    def __repr__(self):             # 好看的打印
        return f'Vector({self.x}, {self.y})'


v1 = Vector(1, 2)
v2 = Vector(3, 4)
print(v1 + v2)           # Vector(4, 6)    ✅
print(v1 == v2)          # False            ✅
```

## 9.4 哈希与集合/字典键

> **铁律：重写 `__eq__` 就必须重写 `__hash__`！**

| 方法 | 作用 | 关联场景 |
|:-----|:-----|:--------|
| `__hash__` | 返回对象的哈希值（整数） | 作为 dict 键 / 放入 set 时需要 |
| `__eq__` | 判断两个对象是否相等 | 配合 `__hash__` 使用 |

```python
class Card:
    def __init__(self, suit, rank):
        self.suit = suit
        self.rank = rank
    
    def __repr__(self):
        return f'{self.suit}{self.rank}'
    
    def __hash__(self):
        return hash((self.suit, self.rank))     # 返回稳定哈希值
    
    def __eq__(self, other):
        return self.suit == other.suit and self.rank == other.rank


c1, c2 = Card("♠", "A"), Card("♠", "A")
c3 = Card("♥", "K")

# 放入 set → 自动去重
my_set = {c1, c2, c3}
print(my_set)              # {♠A, ♥K}      ✅ c1和c2被视为同一个！

# 作为 dict 键
card_count = {c1: 4, c3: 1}
print(card_count[c2])      # 4              ✅ c2能找到c1的值！
```

**原理**：
```
集合去重：
  加入元素 → 算 hash → 定位桶
  桶里已有元素 → 用 eq 比较
  hash相同 + eq相同 → 视为重复，丢弃

字典键：完全一致，只是桶里多存了个 value
```

## 9.5 上下文管理器 — with 语法

| 方法 | 作用 | 触发时机 |
|:-----|:-----|:--------|
| `__enter__` | 进入上下文时执行 | `with` 开始时 |
| `__exit__` | 退出上下文时执行 | `with` 结束时（无论正常还是异常）|

```python
class Timer:
    """计时器上下文管理器"""
    
    import time
    
    def __enter__(self):
        self.start = time.time()
        print("⏱ 计时开始...")
        return self              # 赋给 as 后面的变量
    
    def __exit__(self, exc_type, exc_val, exc_tb):
        elapsed = time.time() - self.start
        print(f"⏱ 计时结束，耗时 {elapsed:.3f} 秒")
        return False             # False=不吞异常，True=已处理


with Timer() as t:
    sum(range(10_000_000))
    # ⏱ 计时开始...
    # ⏱ 计时结束，耗时 0.xxx 秒
```

**实际应用 — 文件自动管理**：

```python
class MyFile:
    def __init__(self, filename, mode):
        self.filename = filename
        self.mode = mode
    
    def __enter__(self):
        self.f = open(self.filename, self.mode)
        return self.f              # as f 拿到的就是这个
    
    def __exit__(self, exc_type, exc_val, exc_tb):
        self.f.close()            # 保证一定会关闭！
        return False


with MyFile('test.txt', 'w') as f:
    f.write('hello')
# 无论中间是否抛异常，__exit__ 都会执行 → 文件一定关闭
```

## 9.6 属性访问拦截

| 方法 | 触发时机 | 用途 |
|:-----|:---------|:-----|
| `__getattr__` | 访问**不存在**的属性时 | 动态代理/转发 |
| `__setattr__` | 设置属性时触发 | 属性校验/拦截 |

```python
class Wrapper:
    def __init__(self, obj):
        self.obj = obj
    
    def __getattr__(self, name):         # 访问不存在的属性
        print(f'⚠ 尝试访问 {name}，转发给内部对象')
        return getattr(self.obj, name)
    
    def __setattr__(self, name, value):   # 设置属性
        if name == 'obj':
            super().__setattr__(name, value)
        else:
            setattr(self.obj, name, value)


w = Wrapper([1, 2, 3])
print(w.append)        # ⚠ 尝试访问 append...  （动态代理!）
w.append(4)            # 实际调用的是内部列表的 append
```

## 9.7 可迭代对象 — iter/next

```python
class Countdown:
    def __init__(self, n):
        self.n = n
    
    def __iter__(self):                  # 返回迭代器自身
        return self
    
    def __next__(self):                  # 每次 for 循环调用一次
        if self.n <= 0:
            raise StopIteration          # 终止信号
        self.n -= 1
        return self.n + 1


for num in Countdown(5):
    print(num, end=' ')                # 1 2 3 4 5
```

## 9.8 可调用对象 — call

```python
class Accumulator:
    def __init__(self):
        self.total = 0
    
    def __call__(self, value):           # 让实例可以像函数一样调用
        self.total += value
        return self.total


acc = Accumulator()
print(acc(10))                # 10    ← 像函数一样调用了！
print(acc(20))                # 30
print(acc(5))                 # 35
```

**应用场景**：装饰器类（`Record.__call__`）、状态机、策略模式。
