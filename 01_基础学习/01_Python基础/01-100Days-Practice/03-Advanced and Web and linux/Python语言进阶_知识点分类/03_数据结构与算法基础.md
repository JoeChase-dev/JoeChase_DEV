## 3.1 算法复杂度（大O标记）

| 复杂度 | 名称 | 典型算法/场景 |
|:------:|:-----|:-------------|
| O(1) | 常量时间 | 布隆过滤器、哈希存储 |
| O(log n) | 对数时间 | 折半查找（二分查找） |
| O(n) | 线性时间 | 顺序查找、计数排序 |
| O(n log n) | 对数线性 | 归并排序、快速排序 |
| O(n²) | 平方时间 | 选择排序、插入排序、冒泡排序 |
| O(n³) | 立方时间 | Floyd算法、矩阵乘法 |
| O(2ⁿ) | 几何级数 | 汉诺塔 |
| O(n!) | 阶乘 | 旅行商问题（NPC） |

## 3.2 排序算法实现

### 选择排序 — O(n²)

```python
def select_sort(items, comp=lambda x, y: x < y):
    """简单选择排序"""
    items = items[:]           # 切片复制，不修改原列表
    for i in range(len(items) - 1):
        min_index = i
        for j in range(i + 1, len(items)):
            if comp(items[j], items[min_index]):
                min_index = j
        items[i], items[min_index] = items[min_index], items[i]
    return items
```

### 冒泡排序 — O(n²)

```python
def bubble_sort(items, comp=lambda x, y: x > y):
    """冒泡排序"""
    items = items[:]
    for i in range(len(items) - 1):
        swapped = False
        for j in range(len(items) - 1 - i):
            if comp(items[j], items[j + 1]):
                items[j], items[j + 1] = items[j + 1], items[j]
                swapped = True
        if not swapped:
            break    # 提前退出：已有序则停止
    return items
```

### 搅拌排序（鸡尾酒排序）— 冒泡升级版

```python
def bubble_sort(items, comp=lambda x, y: x > y):
    """搅拌排序：双向冒泡"""
    items = items[:]
    for i in range(len(items) - 1):
        swapped = False
        # 正向冒泡（左→右）
        for j in range(len(items) - 1 - i):
            if comp(items[j], items[j + 1]):
                items[j], items[j + 1] = items[j + 1], items[j]
                swapped = True
        if swapped:
            swapped = False
            # 反向冒泡（右→左）
            for j in range(len(items) - 2 - i, i, -1):
                if comp(items[j - 1], items[j]):
                    items[j], items[j - 1] = items[j - 1], items[j]
                    swapped = True
        if not swapped:
            break
    return items
```

### 归并排序 — O(n log n)

```python
def merge(items1, items2, comp=lambda x, y: x < y):
    """合并两个有序列表"""
    items = []
    index1, index2 = 0, 0
    while index1 < len(items1) and index2 < len(items2):
        if comp(items1[index1], items2[index2]):
            items.append(items1[index1])
            index1 += 1
        else:
            items.append(items2[index2])
            index2 += 1
    # 拼接剩余部分
    items += items1[index1:]
    items += items2[index2:]
    return items


def merge_sort(items, comp=lambda x, y: x < y):
    """归并排序（公开接口 / Facade Pattern 门面模式）"""
    return _merge_sort(list(items), comp)


def _merge_sort(items, comp):
    """归并排序（内部递归函数，_前缀表示私有）"""
    if len(items) < 2:
        return items
    mid = len(items) // 2
    left = _merge_sort(items[:mid], comp)
    right = _merge_sort(items[mid:], comp)
    return merge(left, right, comp)
```

**命名规范**：
| 前缀 | 含义 | 类比 |
|:-----|:-----|:-----|
| 无前缀 | public — 公共 API | 餐厅的门面 |
| `_` 单下划线 | protected/internal — 内部使用 | 厨房内部 |
| `__` 双下划线 | private — 强制名称改写 | 后厨密室 |

## 3.3 查找算法

### 顺序查找 — O(n)

```python
def seq_search(items, key):
    """顺序查找"""
    for index, item in enumerate(items):   # enumerate 更 Pythonic
        if item == key:
            return index
    return -1
```

### 折半查找（二分查找）— O(log n)

> **前提条件**：序列必须有序

```python
def bin_search(items, key):
    """折半查找"""
    start, end = 0, len(items) - 1
    while start <= end:
        mid = (start + end) // 2
        if key > items[mid]:
            start = mid + 1
        elif key < items[mid]:
            end = mid - 1
        else:
            return mid      # 找到了
    return -1               # 不存在
```
