### 🔘 第5题 — 列表切片
```python
lst = [0, 1, 2, 3, 4, 5]
print(lst[::2])
```
**正确答案: A** ✅ → `[0, 2, 4]`

`lst[::2]` 完整写法是 `lst[start:end:step]`：
- `start` 默认为 0（从头开始）
- `end` 默认为 len(lst)（到结尾）
- `step = 2`（每隔一个取一个）→ 取索引 0, 2, 4

> 🎯 **切片三参数速记**: `lst[起点:终点:步长]`  
> - `lst[::-1]` → 列表反转  
> - `lst[::2]` → 取偶数位置元素  
> - `lst[1::2]` → 取奇数位置元素

### 🔘 第7题 — 集合特性
```python
a = {1, 2, 3}
b = {2, 3, 4}
print(a | b)
```
**正确答案: C** ✅ → `{1, 2, 3, 4}`

`|` 是集合的**并集**运算符，等价于 `a.union(b)`：
- a 中有: `{1, 2, 3}`
- b 中有: `{2, 3, 4}`
- 并集: 所有出现过的元素去重 → `{1, 2, 3, 4}`

> 📋 **集合运算全家桶：**
> | 运算符 | 方法 | 含义 | 示例结果 |
> |--------|------|------|---------|
> | `\|` | `union()` | 并集 | `{1,2,3,4}` |
> | `&` | `intersection()` | 交集 | `{2,3}` |
> | `-` | `difference()` | 差集 | `{1}` |
> | `^` | `symmetric_difference()` | 对称差集 | `{1,4}` |

### 🔘 第8题 — 循环与else ⭐⭐（面试常考）
```python
for i in range(3):
    if i == 2:
        break
    print(i)
else:
    print("done")
```
**正确答案: A** ✅ → 输出 **2行**（只输出 `0\n1`）

**关键知识点：for...else 中的 else 什么时候执行？**

```
for...else 规则：
├── for 循环正常结束（没有被 break 打断）→ else 会执行 ✅
└── for 循环被 break 打断                → else 不执行 ❌
```

本题执行流程：
```
i=0 → if条件不满足 → print(0)     ← 输出第1行
i=1 → if条件不满足 → print(1)     ← 输出第2行
i=2 → if条件满足 → break! 跳出循环  → else被跳过!
最终输出: 0 \n 1 （共2行）
```

> 💡 **实际应用场景**: 用 for...else 来判断循环中是否找到了目标元素
> ```python
> def find(target, lst):
>     for item in lst:
>         if item == target:
>             print(f"找到 {target}!")
>             break
>     else:
>         print(f"{target} 不存在")
> ```

### 🔘 第10题 — 可变默认参数陷阱 ⭐⭐⭐（必考！）
经典面试题！以下代码运行结果是什么？
```python
def add_item(item, lst=[]):
    lst.append(item)
    return lst

print(add_item("a"))   # 第一次调用
print(add_item("b"))   # 第二次调用
```
**正确答案: C** ✅ → `['a']` 和 `['a', 'b']`

**这是Python最经典的坑之一！🔥**

```
问题本质：默认参数 lst=[] 只在【函数定义时】创建一次！

第一次调用 add_item("a"):
├── 使用默认列表 []（此时是空的）
├── 追加 "a" → ['a']
└── 返回 ['a']

第二次调用 add_item("b"):
├── 还是用那个【同一个】默认列表！（它已经不是空的了！）
├── 这个列表现在是 ['a']
├── 追加 "b" → ['a', 'b']  ← 蹊跷！上次的数据还在！
└── 返回 ['a', 'b']

内存中的情况:
函数定义时: lst ──→ [] (地址0x001)
第一次调用后:  lst ──→ ['a'] (还是地址0x001!)
第二次调用后:  lst ──→ ['a','b'] (同一个地址!)
```

✅ **正确的写法**（用 None 作为哨兵值）：
```python
def add_item(item, lst=None):
    if lst is None:
        lst = []
    lst.append(item)
    return lst

print(add_item("a"))  # ['a']
print(add_item("b"))  # ['b']  ← 这次对了！每次都是新列表
```

### 🖥️ 第17题 — 引用 vs 拷贝 ⭐⭐
```python
a = [1, 2, 3]
b = a       # b 和 a 指向同一个列表对象（引用赋值）
b.append(4) # 修改这个共享的对象
print(a)    # [1, 2, 3, 4]  ← a 也变了！因为它们是同一个东西
print(b is a)  # True       ← is 判断是否是同一个对象（同一内存地址）
```

**输出：**
```
[1, 2, 3, 4]
True
```

> ⚠️ **核心概念：可变类型的引用陷阱**
> ```
> a = [1,2,3]     a ──→ [1,2,3] (地址 0x100)
> b = a           b ──→ [1,2,3] (地址 0x100)  ← 同一个！
> b.append(4)     修改 0x100 处的内容
>                 a ──→ [1,2,3,4] (地址 0x100)  ← a也受影响
>
> 解决方案：使用拷贝
> b = a.copy()     浅拷贝（一维列表够用）
> b = a[:]         切片也是浅拷贝
> import copy; b = copy.deepcopy(a)  深拷贝（嵌套列表必须用）
> ```

### 🖥️ 第20题 — 闭包与nonlocal
```python
def outer():
    x = 10
    def inner():
        nonlocal x   # 声明x是外层函数的变量（不是新建局部变量）
        x += 5       # 修改外层x的值
        return x
    return inner()

print(outer())  # 15
```

**执行流程：**
```
outer() 被调用
├── x = 10
├── 定义 inner 函数（闭包，捕获了外层变量 x）
├── 调用 inner()
│   ├── nonlocal x → 关联到外层的 x
│   ├── x += 5 → x 从 10 变成 15
│   └── return 15
└── outer() 返回 15
```

> 📌 **三个关键字对比：**
> | 关键字 | 作用域 | 说明 |
> |--------|--------|------|
> | （无声明） | 局部变量 | 新建一个局部变量 |
> | `global x` | 全局作用域 | 直接操作全局变量 |
> | `nonlocal x` | 外层闭包 | 操作最近一层 enclosing 作用域的变量 |

### 💻 第24题（8分）— 温度转换器 ⭐

**参考答案：**

```python
def celsius_to_fahrenheit(c):
    """
    将摄氏度转换为华氏度
    
    Args:
        c: 摄氏温度（可以是单个数字或数字组成的列表）
    
    Returns:
        华氏温度（同类型输入返回同类型输出）
    
    Raises:
        ValueError: 当输入包含非数字时
    """
    # 处理列表批量转换
    if isinstance(c, (list, tuple)):
        result = []
        for temp in c:
            if not isinstance(temp, (int, float)):
                raise ValueError(f"无效的温度值: {temp}")
            result.append(round(temp * 9 / 5 + 32, 2))
        return result
    
    # 处理单值转换
    if not isinstance(c, (int, float)):
        raise ValueError(f"无效的温度值: {c}，必须是数字")
    
    return round(c * 9 / 5 + 32, 2)


# ======== 测试 ========
if __name__ == "__main__":
    # 单值测试
    print(celsius_to_fahrenheit(0))     # 32.0 (水的冰点)
    print(celsius_to_fahrenheit(100))   # 212.0 (水的沸点)
    print(celsius_to_fahrenheit(-40))   # -40.0 (两温标交汇点)
    
    # 批量测试
    print(celsius_to_fahrenheit([0, 100]))  # [32.0, 212.0]
    print(celsius_to_fahrenheit([20, 25, 30]))  # [68.0, 77.0, 86.0]
    
    # 异常测试（取消注释可测试）
    # print(celsius_to_fahrenheit("abc"))  # 抛出 ValueError
```

**解题要点：**
1. 公式 `F = C × 9/5 + 32` 必须准确
2. 用 `isinstance()` 进行类型检查，区分单值和批量输入
3. `round(result, 2)` 保留两位小数，更美观
4. 文档字符串(docstring)说明函数用途和参数

### 💻 第25题（8分）— 统计字符频率 ⭐⭐

**参考答案：**

```python
import string

def char_count(text):
    """
    统计字符串中各字符的出现频率（忽略大小写，排除空格和标点）
    
    Args:
        text: 待统计的字符串
        
    Returns:
        dict: 按出现次数降序排列的 {字符: 次数} 字典
    """
    # 统一转为小写
    text = text.lower()
    
    # 需要排除的字符：空格 + 标点符号
    exclude_chars = set(string.whitespace + string.punctuation)
    
    freq = {}
    for char in text:
        if char in exclude_chars:
            continue  # 跳过空格和标点
        freq[char] = freq.get(char, 0) + 1
    
    # 按次数降序排序
    sorted_freq = dict(sorted(freq.items(), key=lambda item: item[1], reverse=True))
    
    return sorted_freq


# ======== 测试 ========
if __name__ == "__main__":
    result = char_count("Hello World!")
    print(result)
    # {'l': 3, 'o': 2, 'h': 1, 'e': 1, 'w': 1, 'r': 1, 'd': 1}
    
    # 更复杂的测试
    text2 = "Python编程很有趣！Python是最好的语言。"
    result2 = char_count(text2)
    for char, count in result2.items():
        print(f"'{char}': {count}次")
```

**解题思路拆解：**
```
步骤1: text.lower()          → 统一转小写，忽略大小写差异
步骤2: 构建排除集合           → 空格 + 标点符号全部跳过
步骤3: 遍历计数               → 用 dict.get(char,0)+1 惯用法
步骤4: sorted(...reverse=True) → 降序排列，高频在前
```

### 💻 第26题（12分）— 学生成绩管理（综合项目）⭐⭐⭐

**参考答案：**

```python
"""
学生成绩管理系统 v1.0
功能：添加/查询/统计/查找不及格/排序
数据结构：dict {姓名(str): 成绩(int)}
"""

# 存储所有学生数据的字典
students = {}


def add_student(name, score):
    """添加学生"""
    students[name] = score
    print(f"✅ 已添加: {name} - {score}分")


def query_score(name):
    """查询学生成绩"""
    if name in students:
        print(f"📖 {name}的成绩是: {students[name]}分")
    else:
        print(f"❌ 未找到学生: {name}")


def show_statistics():
    """显示统计信息"""
    if not students:
        print("📭 暂无学生数据")
        return
    
    scores_list = list(students.values())
    avg_score = sum(scores_list) / len(scores_list)
    
    print("\n========== 📊 成绩统计 ==========")
    print(f"📚 学生总数: {len(students)} 人")
    print(f"🏆 最高分: {max(scores_list)} 分")
    print(f"📉 最低分: {min(scores_list)} 分")
    print(f"📐 平均分: {avg_score:.2f} 分")


def find_failed(threshold=60):
    """找出不及格的学生"""
    failed = {name: score for name, score in students.items() if score < threshold}
    
    if failed:
        print(f"\n⚠️ 不及格学生（<{threshold}分）:")
        for name, score in sorted(failed.items(), key=lambda x: x[1]):
            print(f"   ❌ {name}: {score}分")
    else:
        print(f"\n🎉 全部及格！无人低于{threshold}分！")


def rank_students(reverse=True):
    """按成绩排名"""
    if not students:
        print("📭 暂无学生数据")
        return
    
    ranked = sorted(students.items(), key=lambda x: x[1], reverse=reverse)
    
    order = "降序📉" if reverse else "升序📈"
    print(f"\n========== 🏅 成绩排名({order}) ==========")
    for rank, (name, score) in enumerate(ranked, 1):
        medal = ["🥇", "🥈", "🥉"][rank-1] if rank <= 3 else f"{rank}."
        print(f"   {medal} {name}: {score}分")


def show_all():
    """显示所有学生"""
    if not students:
        print("📭 暂无学生数据")
        return
    print("\n========== 📋 全部学生 ==========")
    for name, score in students.items():
        status = "✅" if score >= 60 else "❌"
        print(f"   {status} {name}: {score}分")


# ========== 主菜单 ==========
def main():
    """主程序入口"""
    print("=" * 40)
    print("🎓 学生成绩管理系统 v1.0")
    print("=" * 40)
    
    while True:
        print("""
━━━ 菜单 ━━━
1. ➕ 添加学生
2. 📖 查询成绩
3. 📊 统计信息
4. ⚠️ 查找不及格
5. 🏅 成绩排名
6. 👁 显示全部
7. 👋 退出系统
""")
        
        choice = input("请选择操作 (1-7): ").strip()
        
        if choice == "1":
            name = input("请输入姓名: ").strip()
            try:
                score = int(input("请输入成绩 (0-100): ").strip())
                if 0 <= score <= 100:
                    add_student(name, score)
                else:
                    print("❌ 成绩应在 0-100 之间")
            except ValueError:
                print("❌ 请输入有效的数字成绩")
                
        elif choice == "2":
            name = input("请输入要查询的姓名: ").strip()
            query_score(name)
            
        elif choice == "3":
            show_statistics()
            
        elif choice == "4":
            find_failed()
            
        elif choice == "5":
            rank_students()
            
        elif choice == "6":
            show_all()
            
        elif choice == "7":
            print("👋 感谢使用，再见！")
            break
            
        else:
            print("❌ 无效选项，请重新输入")


if __name__ == "__main__":
    main()
```

**核心设计思路：**

```
┌─────────────────────────────────────┐
│         数据层: students 字典        │
│    {"Alice": 88, "Bob": 92, ...}    │
├─────────────────────────────────────┤
│         功能层: 5个函数               │
│  add / query / stats / fail / rank  │
├─────────────────────────────────────┤
│         交互层: main() 菜单循环       │
│    while + input + if/elif 分发       │
└─────────────────────────────────────┘
```

**涉及的知识点汇总：**
| 知识点 | 应用位置 |
|--------|---------|
| 字典 CRUD | `students[name] = score` / `in` / `.get()` |
| 字典推导式 | `find_failed()` 中过滤不及格 |
| `sorted()` + lambda | `rank_students()` 排序 |
| `enumerate()` | 排名序号生成 |
| `try/except` | 成绩输入异常处理 |
| f-string | 所有格式化输出 |
| `while True` 循环 | 主菜单持续交互 |
| 函数封装 | 每个功能独立函数 |
