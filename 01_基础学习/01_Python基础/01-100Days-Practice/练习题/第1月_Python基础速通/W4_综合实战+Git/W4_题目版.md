### 💻 第6题（25分）— 猜数字游戏 ⭐⭐
**经典入门项目！** 实现一个交互式猜数字游戏：

**功能要求：**
1. 系统随机生成 1-100 的整数
2. 用户输入猜测数字，给出提示："太大了" / "太小了" / "恭喜你猜对了！"
3. 记录猜测次数
4. 猜对后显示总共猜了多少次，并评价表现：
   - ≤5次：🏆 太厉害了！你是神算子！
   - ≤10次：👍 不错不错！
   - >10次：💪 继续加油！

**进阶要求（加分项）：**
5. 游戏结束后询问是否再来一局
6. 输入非法字符（非数字）时不崩溃，给出友好提示
7. 用装饰器记录每次游戏的日志（开始时间/结束时间/结果）

### 💻 第8题（30分）— 个人密码管理器 ⭐⭐⭐⭐
**安全工具开发！** 实现一个简易但实用的密码管理器：

**功能需求：**

1. **密码存储**：
   - 每条记录包含：平台名称、用户名、加密后的密码、创建时间
   - 密码用简单的加密方式保存（如 base64 或自定义移位）
   - 数据存储为 `passwords.json`

2. **CRUD 操作**：
   - `add` — 新增密码条目
   - `list` — 显示所有平台名（隐藏密码）
   - `get <平台>` — 查看指定平台的密码（需要输入主密码验证）
   - `delete <平台>` — 删除条目
   - `search <关键词>` — 模糊搜索

3. **安全特性**：
   - 主密码验证（启动时输入）
   - 密码强度检测（弱/中/强三级评估）
   - 密码生成器：自动生成强随机密码（可配置长度和字符集）

4. **辅助功能**：
   - `generate` — 生成随机密码
   - `strength <密码>` — 评估给定密码的强度
   - `backup` — 备份到带时间戳的文件
