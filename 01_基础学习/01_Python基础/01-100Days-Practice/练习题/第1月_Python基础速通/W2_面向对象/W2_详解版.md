### 🔘 第3题 — 继承与方法重写 ⭐⭐
```python
class Animal:
    def speak(self):        # 父类方法
        return "动物叫"

class Dog(Animal):
    def speak(self):        # 子类重写父类方法
        return "汪汪!"

d = Dog()
print(d.speak())
```
**正确答案: B** ✅ → `汪汪!`

**原理：方法重写(Method Override)**
```
Dog 类继承自 Animal
Dog 定义了自己的 speak() 方法 —— 覆盖了父类的版本
当 d.speak() 被调用时：
  1. Python 先在 Dog 类中找 speak()
  2. 找到了！使用 Dog 自己的版本
  3. 返回 "汪汪!"

如果想调用父类的方法：
  super(Dog, self).speak()  # 或在Python3中简写为: super().speak()
```

### 🔘 第4题 — 多继承MRO（方法解析顺序）⭐⭐⭐
```python
class A:
    def show(self):
        print("A")

class B(A):
    def show(self):
        print("B")
        
class C(A):
    def show(self):
        print("C")

class D(B, C):
    pass    # 没有定义show()

d = D()
d.show()
```
**正确答案: B** ✅ → 输出 `B`

**MRO (Method Resolution Order) 解析：**

Python 使用 **C3线性化算法** 确定多继承时的方法查找顺序：

```
D 的继承关系:  D(B, C)  →  B(A), C(A)

D.__mro__ 查找顺序:
D → B → C → A → object

d.show() 调用过程:
① 先查 D → 没有 show() 
② 再查 B → 有 show()! → 输出 "B" ✅ 停止搜索

可以用代码验证:
>>> D.__mro__
(<class '__main__.D'>, <class '__main__.B'>, <class '__main__.C'>, <class '__main__.A'>, <class 'object'>)
```

> 💡 **MRO 规则（简化记忆）**: 从左到右深度优先，但每个类只出现一次。继承列表中**左边的优先**！

### 🔘 第5章 — 魔术方法 `__str__` vs `__repr__`
```python
class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    def __str__(self):
        return f"({self.x}, {self.y})"

p = Point(3, 4)
print(p)      # 调用 str(p) → 触发 __str__
str(p)        # 直接调用 str() → 也触发 __str__
```
**正确答案: B** ✅ → `(3, 4)` 和 `(3, 4)`

> 📋 **`__str__` vs `__repr__` 对比：**
>
> | 方法 | 触发时机 | 用途 |
> |------|---------|------|
> | `__str__` | `print(obj)`, `str(obj)` | 给**用户看**的信息，友好可读 |
> | `__repr__` | `repr(obj)`, 交互式解释器直接输入变量 | 给**开发者看**的信息，通常可用于 `eval()` 还原 |
>
> ```python
> class Point:
>     def __repr__(self):
>         return f"Point({self.x}, {self.y})"  # 开发者友好
>     
> p = Point(3, 4)
> p          # 在交互式环境中输出: Point(3, 4)
> ```
>
> 如果只定义了 `__repr__`，`print()` 时也会 fallback 到它。建议两个都定义！

### ✏️ 第9题 — 类属性 vs 实例属性 ⭐⭐
```python
class Dog:
    species = "犬类"      # 类属性（存在类上，所有实例共享）
    
    def __init__(self, name):
        self.name = name   # 实例属性（存在每个实例上，独立）

d1 = Dog("旺财")

Dog.species = "修改后的物种"
```

```
Dog (类)

├── d1 (实例)
│   └── name = "旺财"          ← 实例属性（独立的）
│
└── d2 (实例)
    └── name = "大黄"          ← 实例属性（独立的）
    
d1.species 查找链: d1.__dict__没找到 → Dog.__dict__找到了!
```

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
        
class Student(Person):
    def __init__(self, name, age, student_id):
        super().__init__(name, age)   # 第一空：super() 或 super(Student, self)

s = Student("张三", 18, "S20240001")
```

>
> ```python
> # Python 3 简写
> super().__init__(name, age)
>
> # Python 2 写法（兼容旧代码可能会看到）
> super(Student, self).__init__(name, age)

### 🖥️ 第14题 — 三种方法对比 ⭐⭐⭐
```python
class Demo:
    count = 0                    # 类属性：计数器
    
    def __init__(self):
        Demo.count += 1          # 每创建一个实例就+1
    
    @classmethod                 # 类方法：操作类属性
    def get_count(cls):
        return cls.count         # cls 指向类本身（Demo）
    
    @staticmethod                # 静态方法：独立工具函数
    def info():
        return "这是一个Demo类"   # 无 self，无 cls

a = Demo()   # count = 1
b = Demo()   # count = 2
c = Demo()   # count = 3

print(Demo.get_count())   # 3  ← 类方法可以直接通过类名调用
print(Demo.info())        # 这是一个Demo类  ← 静态方法也是
print(a.count)            # 3  ← 实例也可以访问类属性
```

**输出：**
```
3
这是一个Demo类
3
```

> 📊 **三种方法核心区别：**
>
> | 类型 | 装饰器 | 第一个参数 | 能访问什么 | 调用方式 |
> |------|--------|----------|-----------|---------|
> | 实例方法 | 无 | `self` | 实例属性 + 类属性 | `obj.method()` |
> | 类方法 | `@classmethod` | `cls`(指类) | 类属性 + 类方法 | `Class.method()` / `obj.method()` |
> | 静态方法 | `@staticmethod` | 无 | 什么都访问不到（除非传参） | `Class.method()` / `obj.method()` |

### 🖥️ 第15题 — property 装饰器 ⭐⭐
```python
class BankAccount:
    def __init__(self, balance):
        self._balance = balance    # 单下划线：约定"内部使用"，但不强制
    
    @property                      # 将方法变成"只读属性"
    def balance(self):
        return self._balance
    
    @balance.setter               # 设置属性时的验证逻辑
    def balance(self, value):
        if value < 0:
            raise ValueError("余额不能为负!")
        self._balance = value

acct = BankAccount(1000)
print(acct.balance)             # 1000  通过@property读取（像访问普通属性一样）

acct.balance = 2000             # 通过@balance.setter修改
print(acct._balance)            # 2000  直接访问底层变量

acct.balance = -500             # 触发验证逻辑！< 0 → 抛异常
```

**输出：**
```
1000
2000
ValueError: 余额不能为负!
```

> 💡 **property 的好处**：对外提供简洁的接口（像访问属性一样），内部控制验证逻辑。这就是**封装**的核心思想！  
> 📌 下划线命名规范：`_var`（保护）、`__var`（名称改写/更严格的私有）

### 🖥️ 第16题 — 多态的实际应用 ⭐⭐
```python
class Shape:
    def area(self):
        pass    # 父类定义接口（抽象概念），子类各自实现

class Rectangle(Shape):
    def __init__(self, w, h):
        self.w = w
        self.h = h
    def area(self):
        return self.w * self.h

class Circle(Shape):
    def __init__(self, r):
        self.r = r
    def area(self):
        return 3.14 * self.r ** 2

shapes = [Rectangle(3, 4), Circle(5)]
for s in shapes:
    print(f"{s.__class__.__name__}: {s.area()}")
    # s.__class__.__name__ 获取类名字符串
    # s.area() 根据实际类型调用对应的方法（多态！）
```
**输出：**
```
Rectangle: 12
Circle: 78.5
```

> 🎯 **多态的核心思想**：同一个接口（`area()`），不同的实现。你不需要知道具体是什么形状，只要调用 `.area()` 就能得到正确的面积。这让代码更加灵活和可扩展！

### 🖥️ 第17题 — try/except/finally 执行流程
```python
def divide(a, b):
    try:
        return a / b          # 正常执行 → 返回值被暂存
    except ZeroDivisionError:
        print("⚠️ 被除数为0!")
        return None
    finally:
        print("🧹 清理工作完成...")  # 无论是否异常，finally必定执行！

print(divide(10, 2))    # 正常情况
print("---")
print(divide(10, 0))    # 异常情况
```

**输出：**
```

5.0
---
⚠️ 被除数为0!
🧹 清理工作完成...
None
```

> ⚠️ **关键点：finally 总是最后执行！** 即使前面有 return，也要先执行 finally 块中的代码，然后才返回。这就是为什么 finally 常用来做资源清理工作。

### 🖥️ 第19题 — 文件逐行处理 + 数据聚合
假设 `scores.txt` 内容为：
```
Alice,88
Bob,92
Charlie,78
```

```python
results = []
with open("scores.txt", "r") as f:     # with 自动管理文件关闭
    for line in f:                     # 逐行读取（内存友好！）
        line = line.strip()            # 去掉换行符和空白
        if line:                       # 跳过空行
            name, score = line.split(",")  # 按逗号分割
            results.append((name, int(score)))  # 存为元组列表

total = sum(score for _, score in results)  # 生成器表达式求和
avg = total / len(results)
```
**输出：**
```
共有 3 名学生
总分: 258, 平均分: 86.0
```

**逐步计算：**
```
results = [("Alice", 88), ("Bob", 92), ("Charlie", 78)]
total = 88 + 92 + 78 = 258
avg = 258 / 3 = 86.0
```

> 💡 **为什么用逐行读取而不是 `f.read().splitlines()`？**  
> 当文件很大时（如日志文件几百MB），逐行读取不会一次性把整个文件加载到内存中，非常节省内存！这是**流式处理**的思想。

```python
        self.title = title
        self.author = author
        self.isbn = isbn
        self._is_borrowed = False  # 默认未借出
    
    @property
    def is_borrowed(self) -> bool:
        """获取借阅状态（只读）"""
        return self._is_borrowed
    
    def borrow(self):
        """标记为已借出"""
        if self._is_borrowed:
            raise BookAlreadyBorrowedError(f"'{self.title}' 已被借出")
        self._is_borrowed = True
        return True
    
    def return_book(self):
        """归还书籍"""
        if not self._is_borrowed:
            raise BookNotFoundError(f"'{self.title}' 未被借出，无法归还")
        self._is_borrowed = False
        return True
    
    def __str__(self):
        status = "📕 已借出" if self._is_borrowed else "📗 在馆"
        return f"[{status}] 《{title}》 - {author} (ISBN:{isbn})".format(
            title=self.title, author=self.author, isbn=self.isbn
```

```python
"""
Logger 日志记录器
支持同时输出到控制台 + 文件
"""

import os
from datetime import datetime


class Logger:
    """多功能日志记录器"""
    
    LEVELS = {
        "DEBUG": "\033[36m",     # 青色
        "INFO": "\033[32m",      # 绿色
        "WARNING": "\033[33m",   # 黄色
        "ERROR": "\033[31m",     # 红色
    }
    RESET = "\033[0m"  # 重置颜色
    
    def __init__(self, log_dir="logs", name="app"):
        self.log_dir = log_dir
        self.name = name
        self._ensure_dir(log_dir)
    
    def _ensure_dir(self, directory):
        """确保目录存在"""
        if not os.path.exists(directory):
            os.makedirs(directory)
    
    def _get_log_file(self):
        """获取今天的日志文件路径"""
        today = datetime.now().strftime("%Y-%m-%d")
        return os.path.join(self.log_dir, f"{today}.log")
    
    def _format_message(self, level, message):
        """格式化日志消息"""
        timestamp = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
        color = self.LEVELS.get(level, "")
        return f"{color}[{timestamp}] [{level:>8}] {message}{self.RESET}"
    
    def log(self, level, message):
        """记录日志（同时输出到控制台和文件）"""
        formatted = self._format_message(level, message)
        
        # 控制台输出（带颜色）
        print(formatted)
        
        # 文件输出（纯文本无颜色）
        plain_text = f"[{datetime.now().strftime('%Y-%m-%d %H:%M:%S')}] [{level}] {message}\n"
        log_file = self._get_log_file()
        
        with open(log_file, "a", encoding="utf-8") as f:
            f.write(plain_text)
    
    # ======== 便捷方法 ========
    def debug(self, message):
        self.log("DEBUG", message)
    
    def info(self, message):
        self.log("INFO", message)
    
    def warning(self, message):
        self.log("WARNING", message)
    
    def error(self, message):
        self.log("ERROR", message)
```

```python
import json

class StudentManager:

    def __init__(self, data_file=DATA_FILE):
        self.data_file = data_file
        self.students = {}
        self._load()  # 启动时自动加载数据

        """从JSON文件加载数据"""
        if not os.path.exists(self.data_file):
            print(f"🆕 首次启动，创建新的数据文件")
            self.students = {}
            self._save()
            return
        
        try:
            with open(self.data_file, "r", encoding="utf-8") as f:
                self.students = json.load(f)
            print(f"📂 已加载 {len(self.students)} 名学生的数据")

            print(f"⚠️ 数据文件损坏！将从空数据开始")
            self.students = {}
        except Exception as e:
            print(f"⚠️ 加载数据失败: {e}")
            self.students = {}
    
    def _save(self):
        """将数据保存到JSON文件"""
        try:
            with open(self.data_file, "w", encoding="utf-8") as f:
                json.dump(self.students, f, ensure_ascii=False, indent=2)
        except IOError as e:
            print(f"❌ 保存失败: {e}")
    
    # ========== CRUD 操作（每次修改后自动保存）==========

        if name in self.students:
            print(f"❌ 学生 {name} 已存在")

            print("❌ 成绩必须是0-100之间的数字")
            return False
        
        self.students[name] = score
        self._save()  # 自动保存！
        print(f"✅ 已添加: {name} ({score}分)")
        return True

            del self.students[name]

        else:
            print(f"❌ 未找到: {name}")
```

```python
def char_frequency(text: str, ignore_case=True) -> dict:
    """字符频率统计"""
    if ignore_case:
        text = text.lower()
    freq = {}
    for char in text:
        freq[char] = freq.get(char, 0) + 1
    return dict(sorted(freq.items(), key=lambda x: x[1], reverse=True))

    """蛇形转驼峰命名"""

    """截断长文本"""
    if len(text) <= max_length:
        return text
    return text[:max_length - len(suffix)] + suffix
```

```python
"""数学工具函数集"""
import math


def factorial(n: int) -> int:
    """阶乘 n!"""
    if n < 0:
        raise ValueError("负数没有阶乘")
    if n <= 1:
        return 1
    result = 1
    for i in range(2, n + 1):
        result *= i
    return result


def fibonacci(n: int) -> list:
    """斐波那契数列前n项"""
    if n <= 0:
        return []
    if n == 1:
        return [0]
    
    fibs = [0, 1]
    while len(fibs) < n:
        fibs.append(fibs[-1] + fibs[-2])
    return fibs


def is_prime(n: int) -> bool:
    """判断质数"""
    if n < 2:
        return False
    if n == 2:
        return True
    if n % 2 == 0:
        return False
    for i in range(3, int(math.sqrt(n)) + 1, 2):
        if n % i == 0:
            return False
    return True


def gcd(a: int, b: int) -> int:
    """最大公约数（辗转相除法）"""
    while b:
        a, b = b, a % b
    return abs(a)


def lcm(a: int, b: int) -> int:
    """最小公倍数"""
    return abs(a * b) // gcd(a, b)
```

```python
"""文件工具函数集"""
import os
import shutil
from datetime import datetime


def read_file_safe(filepath: str, encoding="utf-8") -> str:
    """安全读取文件（文件不存在返回空字符串而非崩溃）"""
    if not os.path.exists(filepath):
        return ""
    with open(filepath, "r", encoding=encoding) as f:
        return f.read()


def write_file_safe(filepath: str, content: str, encoding="utf-8"):
    """安全写入（自动创建目录）"""
    directory = os.path.dirname(filepath)
    if directory and not os.path.exists(directory):
        os.makedirs(directory)
    with open(filepath, "w", encoding=encoding) as f:
        f.write(content)


def backup_file(filepath: str) -> str:
    """备份文件（添加时间戳后缀）"""
    if not os.path.exists(filepath):
        raise FileNotFoundError(f"文件不存在: {filepath}")
    
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    backup_path = f"{filepath}.{timestamp}.bak"
    shutil.copy2(filepath, backup_path)
    return backup_path


def get_file_info(filepath: str) -> dict:
    """获取文件基本信息"""
    stat = os.stat(filepath)
    return {
        "name": os.path.basename(filepath),
        "size_bytes": stat.st_size,
        "size_human": format_size(stat.st_size),
        "modified": datetime.fromtimestamp(stat.st_mtime).strftime("%Y-%m-%d %H:%M:%S"),
    }


def format_size(size_bytes: int) -> str:
    """将字节转为人类可读的大小"""
    for unit in ["B", "KB", "MB", "GB"]:
        if size_bytes < 1024:
            return f"{size_bytes:.1f}{unit}"
        size_bytes /= 1024
    return f"{size_bytes:.1f}TB"


def list_files(directory: str, extension=None) -> list:
    """列出目录下的文件"""
    files = []
    for item in os.listdir(directory):
        full_path = os.path.join(directory, item)
        if os.path.isfile(full_path):
            if extension is None or item.endswith(extension):
                files.append(full_path)
    return sorted(files)
```
