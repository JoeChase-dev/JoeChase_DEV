## 1.1 推导式（Comprehension）

### 基本语法

```python
# 字典推导式 - 从现有字典筛选并生成新字典
prices = {
    'AAPL': 191.88, 'GOOG': 1186.96, 'IBM': 149.24,
    'ORCL': 48.44, 'ACN': 166.89, 'FB': 208.09, 'SYMC': 21.29
}

# 筛选 value > 100 的条目，生成新字典
prices2 = {key: value for key, value in prices.items() if value > 100}
print(prices2)
# 输出: {'AAPL': 191.88, 'GOOG': 1186.96, 'IBM': 149.24, 'ACN': 166.89, 'FB': 208.09}
```

### 三种推导式对比

| 类型 | 语法 | 示例 | 返回类型 |
|------|------|------|---------|
| **列表推导式** | `[expr for x in iterable if cond]` | `[x**2 for x in range(10)]` | `list` |
| **字典推导式** | `{k: v for k, v in dict.items()}` | `{x: x**2 for x in range(5)}` | `dict` |
| **集合推导式** | `{expr for x in iterable}` | `{x % 3 for x in range(10)}` | `set` |

> 💡 **知识点**：推导式的核心价值在于**声明式编程**——描述"我要什么"而非"怎么做到"，代码更简洁、更 Pythonic。

## 1.2 嵌套列表的坑 ⚠️ 易错

> 这是 Python 初学者最常踩的坑之一！

### ❌ 错误写法 —— 乘法复制引用

```python
a = [[None] * 3] * 3
print(a)
# [[None, None, None], [None, None, None], [None, None, None]]

a[0][0] = 100
print(a)
# [[100, None, None], [100, None, None], [100, None, None]]
#          ↑ 三行同时被修改了！因为三个子列表是同一个对象！
```

**原理图解**：

```
❌ 错误: [None]*3 * 3

   a ──→ [list_0]
            ↓ (乘法只复制引用!)
         list_0 ──→ [100, None, None]
                      ↗ (同一对象)
         list_0 ←───┘
```

### ✅ 正确写法 —— 列表推导式独立创建

```python
score_list = [[None] * len(courses) for _ in range(len(names))]
# 每次循环都创建一个新列表对象，互相独立！
```

**原理图解**：

```
✅ 正确: [[None]*3 for _ in range(3)]

   a ──→ [list_0, list_1, list_2]
            ↓        ↓        ↓
         独立对象  独立对象  独立对象
```

> 📌 **记忆口诀**：**嵌套列表用推导，乘法复制是坑路**

## 1.3 heapq 堆模块

### 核心功能

`heapq` 提供堆队列算法（优先队列），底层是小顶堆结构。

| 函数 | 说明 | 时间复杂度 |
|------|------|-----------|
| `heapq.nlargest(n, iter)` | 取最大的 n 个元素 | O(n log k) |
| `heapq.nsmallest(n, iter)` | 取最小的 n 个元素 | O(n log k) |
| `heapq.heappush(heap, item)` | 入堆 | O(log n) |
| `heapq.heappop(heap)` | 弹出最小值 | O(log n) |
| `heapq.heapify(list)` | 原地转为堆 | O(n) |

### 实际示例

```python
import heapq

# 基础用法：从列表中找最大/最小的 N 个数
nums = [34, 25, 18, 87, 76, 95, 48, 65, 28, 14]

print(heapq.nlargest(3, nums))   # [95, 87, 76]  — 前3大
print(heapq.nsmallest(3, nums))  # [14, 18, 25]  — 前3小
```

```python
# 高级用法：配合 key + lambda 处理复杂数据
data = [
    {'name': 'IBM',   'shares': 100, 'price': 91.1},
    {'name': 'AAPL',  'shares': 50,  'price': 543.22},
    {'name': 'FB',    'shares': 200, 'price': 21.09},
    {'name': 'HPQ',   'shares': 35,  'price': 31.75},
    {'name': 'YHOO',  'shares': 45,  'price': 16.35},
    {'name': 'ACME',  'shares': 75,  'price': 115.65}
]

# key 参数告诉 Python "用 price 字段来比较"
print(heapq.nlargest(2, data, key=lambda x: x['price']))
# [{'name': 'AAPL', ...}, {'name': 'ACME', ...}]  — 价格最高的两个

print(heapq.nsmallest(2, data, key=lambda x: x['price']))
# [{'name': 'YHOO', ...}, {'name': 'FB', ...}]     — 价格最低的两个
```

> 📌 **知识点**：`lambda x: x['price']` 是匿名函数，等价于：
> ```python
> def get_price(x):
>     return x['price']
> ```

## 1.4 itertools 迭代工具库

> Python 的"瑞士军刀"，提供大量高效迭代器工具。

### 核心函数速查表

| 函数 | 功能 | 示例输入 | 输出示例 |
|------|------|---------|---------|
| `permutations(iter, r)` | 全排列（有序） | `'ABC', 2` | `('A','B'), ('A','C'), ('B','A')...` (6个) |
| `combinations(iter, r)` | 组合（无序） | `'ABC', 2` | `('A','B'), ('A','C'), ('B','C')` (3个) |
| `product(*iters)` | 笛卡尔积 | `'AB', '12'` | `('A','1'), ('A','2'), ('B','1'), ('B','2')` |
| `cycle(iter)` | 无限循环 | `'ABC'` | `A B C A B C A ...` (无限!) |

### 详细示例

```python
import itertools

# === permutations: 全排列 (有序，AB ≠ BA) ===
print("=== 全排列 ABC 取2个 ===")
for perm in itertools.permutations('ABCD', 2):
    print(perm)
# ('A','B') ('A','C') ('A','D') ('B','A') ('B','C') ...
# 共 12 = P(4,2) = 4×3 个

print("\n=== 组合 ABCDE 选2个 (无序, AB=BA) ===")
for comb in itertools.combinations('ABCDE', 2):
    print(comb)
# ('A','B') ('A','C') ('A','D') ('A','E') ('B','C') ...
# 共 10 = C(5,2) 个

print("\n=== 笛卡尔积 (所有可能的配对) ===")
for prod in itertools.product('ABCD', '123'):
    print(prod)
# ('A','1') ('A','2') ('A','3') ('B','1') ... 共 12 个
# 等价于嵌套循环:
#   for x in 'ABCD':
#       for y in '123':
#           print((x, y))
```

```python
# === cycle: 无限循环迭代器 ===
counter = 0
for item in itertools.cycle(('A', 'B', 'C')):
    print(item, end=' ')
    counter += 1
    if counter >= 10:
        break
# 输出: A B C A B C A B C A
```

> 📌 **知识点**：`itertools` 的函数返回的是**迭代器（惰性计算）**，不占用额外内存。只有 `for` 循环遍历时才逐个生成值。

## 1.5 collections 集合增强

### 五大常用工具类

| 类名 | 功能 | 使用场景 |
|------|------|---------|
| `namedtuple` | 命名元组（带字段名的不可变对象） | 替代简单类，轻量数据容器 |
| `deque` | 双端队列（头尾插入删除都是 O(1)） | 队列/栈/BFS/滑动窗口 |
| `Counter` | 计数器（统计元素出现频率） | 词频统计/Top-K 问题 |
| `OrderedDict` | 有序字典（保持插入顺序） | Python 3.7+ 的 dict 已内置此特性 |
| `defaultdict` | 带默认值的字典（避免 KeyError） | 分组统计/图邻接表 |

### Counter 实战示例

```python
from collections import Counter

words = [
    'look', 'into', 'my', 'eyes', 'look', 'into', 'my', 'eyes',
    'the', 'eyes', 'the', 'eyes', 'the', 'eyes', 'not', 'around',
    'the', 'eyes', "don't", 'look', 'around', 'the', 'eyes',
    'look', 'into', 'my', 'eyes', "you're", 'under'
]

counter = Counter(words)

# most_common(n): 获取出现频率最高的 n 个元素
print(counter.most_common(3))
# [('eyes', 8), ('the', 5), ('look', 4)]

# 单独查看某个词的频率
print(f"'eyes' 出现 {counter['eyes']} 次")  # 8 次
```

### deque vs list 性能对比

| 操作 | `list` (基于数组) | `deque` (基于双向链表) |
|------|:-:|:-:|
| `append()` / `pop()` 尾部操作 | **O(1)** | **O(1)** |
| `insert(0, x)` / `popleft()` 头部操作 | **O(n)** 需要移动全部元素 | **O(1)** |
| 中间随机访问 `arr[i]` | **O(1)** | **O(n)** |

> 📌 **结论**：需要频繁在**头部**操作时用 `deque`；主要在尾部操作且需随机访问时用 `list`。
