### 🔘 第1题 — Git基本概念
**正确答案: D** ❌ → "git push 直接将工作区的文件推送到远程仓库，不需要先 commit"

| 选项 | 判断 | 解析 |
|------|------|------|
| A | ✅ | Git是分布式的——每个人的电脑上都有完整的仓库（含完整历史） |
| B | ✅ | `git add` 确实是把文件从工作区放到暂存区 |
| C | ✅ | `git commit` 把暂存区内容提交到本地仓库，生成一个commit记录 |
| **D** | **❌ 错误** | **push之前必须先 commit！** push 只能推送已提交的代码。未跟踪或未提交的修改不会被推送 |

> 📌 **Git 三区域模型：**
>
> ```
> 工作区(Working Dir) ──git add──→ 暂存区(Stage) ──git commit──→ 本地仓库(Repo)
>    (你的文件夹)                    (临时存放)              (历史记录)
>                                                          │
>                                                   git push │
>                                                          ↓
>                                                    远程仓库(Remote)
>                                                     (GitHub)
> ```

### 🔘 第2题 — 命令匹配
**正确答案: D** ✅ → ①-D, ②-C, ③-A, ④-B, ⑤-E

| 操作 | 正确命令 | 说明 |
|------|---------|------|
| ①查看状态 | `git status` (D) | 显示哪些文件被修改、哪些在暂存区 |
| ②提交更改 | `git commit -m` (C) | `-m` 后跟提交信息，必填！ |
| ③查看历史 | `git log --oneline` (A) | `--oneline` 让每条记录只占一行 |
| ④关联远程 | `git remote add origin` (B) | `origin` 是默认的远程仓库名称 |
| ⑤创建分支 | `git branch <name>` (E) | 创建但不切换；切换用 `checkout` 或 `switch` |

### 🔘 第3题 — .gitignore
**正确答案: B** ✅ → "*.log 会忽略所有 .log 文件"

| 选项 | 解析 |
|------|------|
| A | ❌ `.gitignore` 可以放在子目录中（规则作用于该目录及子目录），但通常放在根目录管理全局 |
| **B** | **✅** `*.log` 是标准的 glob 模式，匹配所有以 `.log` 结尾的文件 |
| C | ❌ 已被跟踪的文件需要先用 `git rm --cached <file>` 取消跟踪，`.gitignore` 对已跟踪文件无效 |
| D | ❌ `!` 是否定前缀。`!.important.log` 的含义是：**不忽略** important.log（即使前面有 `*.log` 规则） |

.env                  # 环境变量（含密钥！）

.vscode/              # 编辑器配置
.DS_Store             # macOS系统文件

### 🔘 第4题 — 提交流程
**正确答案: B** ✅ → 修改文件 → git add → git commit → git push

**四步曲记忆口诀：改→加→交→推**

```
1. 修改文件         在编辑器里写代码
2. git add .        放入暂存区（"我准备好这些了"）
3. git commit -m "" 提交到本地仓库（"正式记录这个版本"）
4. git push         推送到 GitHub（"分享给全世界"）
```

### 🔘 第5题 — 首次推送步骤
**正确答案: D** ✅ → ③④⑤②①

正确的首次推送流程：
1. **③ 先在 GitHub 上创建空仓库**（网页操作）
2. **④ 本地初始化**: `git init`
3. **⑤ 添加并提交**: `git add . && git commit -m "init"`
4. **② 关联远程仓库**: `git remote add origin <url>`
5. **① 推送**: `git push -u origin main`

> 💡 `-u` 参数的含义：设置上游分支(upstream)。设置后以后直接用 `git push` 就行，不用每次写 `git push origin main`。

"""
猜数字游戏 v1.0
综合运用: random模块 / 循环 / 条件判断 / 异常处理 / 装饰器
"""

import random

from datetime import datetime


# ========== 日志装饰器 ==========
def game_logger(func):
    """记录游戏的开始时间、结束时间和结果"""
    @wraps(func)
    def wrapper(*args, **kwargs):
        start = datetime.now()
        print(f"\n🎮 游戏开始于: {start.strftime('%H:%M:%S')}")
        print("=" * 35)
        
        result = func(*args, **kwargs)
        
        end = datetime.now()
        elapsed = (end - start).total_seconds()
        
        print("=" * 35)
        print(f"🏁 游戏结束于: {end.strftime('%H:%M:%S')} (耗时{elapsed:.1f}秒)")
        
        return result
    return wrapper


# ========== 核心游戏逻辑 ==========
@game_logger
def play_one_round():
    """进行一轮猜数字游戏"""
    
    # 随机生成 1-100 的整数
    secret = random.randint(1, 100)
    attempts = 0
    
    print("🤔 我已经想好了一个 1-100 之间的整数")
    print("来猜猜看吧！（输入 q 退出本轮）\n")
    
    while True:
        user_input = input(f"第{attempts+1}次猜测 > ").strip()
        
        # 退出检测
        if user_input.lower() == 'q':
            print(f"😢 你放弃了...答案是 {secret}")
            return {"result": "quit", "answer": secret}
        
        # 输入验证
        try:
            guess = int(user_input)
        except ValueError:
            print("⚠️ 请输入有效的数字！(1-100之间的整数)\n")
            continue
        
        # 范围检查
        if guess < 1 or guess > 100:
            print("⚠️ 数字必须在 1 到 100 之间！\n")
            continue
        
        attempts += 1
        
        # 判断大小
        if guess < secret:
            hint = f"{guess} 太小了! 🔼 再大一点"

        elif guess > secret:
            hint = f"{guess} 太大了! 🔽 小一点"
            emoji = "🥵"
        else:
            # 猜对了!
            print(f"\n🎉🎉🎉 恭喜你! 答案就是 {secret}!")

                rating = "🏆 太厉害了！你是神算子！"

                rating = "👍 不错不错！很有天赋！"
            else:
                rating = "💪 继续加油！多练练会更好的~"
            
            print(f"   一共猜了 {attempts} 次 | {rating}")
            
            return {
                "result": "win",
                "attempts": attempts,
                "answer": secret,
                "rating": rating,
            }
        
        print(f"  {emoji} {hint}\n")


# ========== 主程序 ==========
def main():
    """
    主循环：支持多轮游戏
    """
    total_games = 0
    wins = 0
    history = []
    
    print("\n" + "=" * 40)
    print("  🎲 猜数字游戏 v1.0")
    print("=" * 40)
    print("  规则: 系统随机一个1-100的数，你来猜！")
    print("  输入 q 可随时退出当前轮\n")

                    if win_attempts:

                print("\n感谢游玩! 下次再见! 👋")

            else:
                print("请输入 y 或 n")


if __name__ == "__main__":

"""
命令行记事本 - NoteCLI v1.0
综合运用: 类设计 / JSON持久化 / 异常处理 / 字符串操作 / 正则搜索
"""

import json
import os
import re
import uuid
from datetime import datetime
from typing import List, Optional, Dict


class Note:
    """单条笔记"""
    
    def __init__(self, title: str, content: str):
        self.id = str(uuid.uuid4())[:8]       # 短ID
        self.title = title.strip()
        self.content = content.strip()
        self.created_at = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
        self.updated_at = self.created_at
    
    def to_dict(self) -> dict:
        return {
            "id": self.id,
            "title": self.title,
            "content": self.content,
            "created_at": self.created_at,
            "updated_at": self.updated_at,
        }
    
    @classmethod
    def from_dict(cls, data: dict) -> 'Note':
        note = cls(data["title"], data["content"])
        note.id = data.get("id", note.id)
        note.created_at = data.get("created_at", note.created_at)
        note.updated_at = data.get("updated_at", note.updated_at)
        return note
    
    def __str__(self):
        return f"[#{self.id}] {self.title} ({self.updated_at})"
    
    def matches_keyword(self, keyword: str) -> bool:
        """检查笔记是否包含关键词"""
        keyword_lower = keyword.lower()
        return (
            keyword_lower in self.title.lower() 
            or keyword_lower in self.content.lower()
        )


class Notebook:
    """记事本管理器"""
    
    def __init__(self, data_file="notes.json"):
        self.data_file = data_file
        self.notes: List[Note] = []
        self._load()
    
    def _load(self):
        """从JSON加载笔记数据"""
        if not os.path.exists(self.data_file):
            return
        
        try:
            with open(self.data_file, "r", encoding="utf-8") as f:
                data = json.load(f)
            self.notes = [Note.from_dict(item) for item in data]
        except (json.JSONDecodeError, KeyError, TypeError) as e:
            print(f"⚠️ 数据文件损坏: {e}，将从头开始")
            self.notes = []
    
    def _save(self):
        """保存到JSON文件"""
        data = [note.to_dict() for note in self.notes]
        with open(self.data_file, "w", encoding="utf-8") as f:
            json.dump(data, f, ensure_ascii=False, indent=2)
    
    # ========== CRUD ==========
    
    def add(self, title: str, content: str) -> Note:
        """添加笔记"""
        note = Note(title, content)
        self.notes.append(note)
        self._save()
        print(f"✅ 笔记已创建: #{note.id} {note.title}")
        return note
    
    def list_all(self) -> None:
        """列出所有笔记"""
        if not self.notes:
            print("📭 还没有笔记。用 'add \"标题\" \"内容\"' 来创建第一篇！")
            return
        
        print(f"\n{'='*50}")
        print(f"📓 共 {len(self.notes)} 篇笔记")
        print('='*50)
        
        for i, note in enumerate(self.notes, 1):
            content_preview = note.content[:20].replace('\n', ' ')
            if len(note.content) > 20:
                content_preview += "..."
            print(f"  {i:>2}. [{note.id}] {note.title}")
            print(f"      📝 {content_preview}")
            print(f"      🕐 {note.updated_at}")
    
    def view(self, index: int) -> Optional[Note]:
        """查看笔记详情"""
        try:
            note = self.notes[index - 1]
        except IndexError:
            print(f"❌ 找不到第 {index} 条笔记（总共只有 {len(self.notes)} 条）")
            return None
        
        print(f"\n{'='*50}")
        print(f"📄 笔记详情")
        print('='*50)
        print(f"  ID:      {note.id}")
        print(f"  标题:    {note.title}")
        print(f"  创建时间: {note.created_at}")
        print(f"  更新时间: {note.updated_at}")
        print(f"\n  内容:")
        print("-" * 46)
        for line in note.content.split("\n"):
            print(f"  {line}")
        print("-" * 46)
        return note
    
    def delete(self, index: int) -> bool:
        """删除笔记"""
        try:
            removed = self.notes.pop(index - 1)
            self._save()
            print(f"🗑 已删除: [{removed.id}] {removed.title}")
            return True
        except IndexError:
            print(f"❌ 找不到第 {index} 条笔记")
            return False
    
    def search(self, keyword: str) -> List[Note]:
        """搜索笔记"""
        results = [n for n in self.notes if n.matches_keyword(keyword)]
        
        if not results:
            print(f"🔍 未找到包含 '{keyword}' 的笔记")
            return []
        
        print(f"\n🔍 搜索 '{keyword}' 找到 {len(results)} 条结果:\n")
        for i, note in enumerate(results, 1):
            print(f"  {i}. [{note.id}] {note.title}")
            # 高亮关键词
            highlighted = re.sub(
                f"({re.escape(keyword)})",
                r"**\1**",
                note.content[:80],
                flags=re.IGNORECASE
            )
            print(f"     ...{highlighted}...")
        
        return results
    
    def export(self, export_path=None) -> str:
        """导出为JSON备份"""
        if export_path is None:
            timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
            export_path = f"notes_backup_{timestamp}.json"
        
        with open(export_path, "w", encoding="utf-8") as f:
            json.dump([n.to_dict() for n in self.notes], f, ensure_ascii=False, indent=2)
        
        print(f"💾 已导出 {len(self.notes)} 条笔记到: {export_path}")
        return export_path


# ========== CLI交互界面 ==========
def parse_command(raw_input: str) -> tuple:
    """
    解析用户命令为 (command, args)
    支持引号包裹的多词参数
    """
    raw_input = raw_input.strip()
    if not raw_input:
        return ("", [])
    
    parts = []
    current = ""
    in_quotes = False
    quote_char = None
    
    for char in raw_input:
        if char in '"\'' and not in_quotes:
            in_quotes = True
            quote_char = char
        elif char == quote_char and in_quotes:
            in_quotes = False
            quote_char = None
        elif char == " " and not in_quotes:
            if current:
                parts.append(current)
                current = ""
        else:
            current += char
    
    if current:
        parts.append(current)
    
    if not parts:
        return ("", [])
    
    command = parts[0].lower()
    args = parts[1:]
    return command, args


def show_help():
    """显示帮助信息"""
    help_text = """
╔════════════════════════════════════════╗
║     📓 NoteCLI 记事本 - 使用帮助       ║
╠════════════════════════════════════════╣
║  命令                      功能       ║
╠════════════════════════════════════════╣
║  add "标题" "内容"         添加笔记    ║
║  list                     列出全部    ║
║  view <编号>              查看详情    ║
║  delete <编号>            删除笔记    ║
║  search <关键词>          搜索笔记    ║
║  export                  导出备份    ║
║  help                     显示帮助    ║
║  exit / quit              退出程序    ║
╚════════════════════════════════════════╝
"""
    print(help_text)


def main():
    """主入口"""
    notebook = Notebook()
    
    print("\n" + "=" * 42)
    print("  📓 NoteCLI - 命令行记事本 v1.0")
    print("=" * 42)
    print('  输入 help 查看帮助 | 输入 exit 退出')
    print(f"  数据文件: {notebook.data_file}")
    
    while True:
        try:
            raw_cmd = input("\n❯ ").strip()
            
            if not raw_cmd:
                continue
            
            cmd, args = parse_command(raw_cmd)
            
            if cmd in ('exit', 'quit', 'q'):
                print("\n👋 再见! 笔记已自动保存.")
                break
            
            elif cmd in ('help', '?', 'h'):
                show_help()
            
            elif cmd == 'add':
                if len(args) >= 2:
                    notebook.add(args[0], " ".join(args[1:]))
                else:
                    print('用法: add "标题" "内容"')
            
            elif cmd == 'list':
                notebook.list_all()
            
            elif cmd == 'view':
                if len(args) >= 1:
                    notebook.view(int(args[0]))
                else:
                    print("用法: view <编号>")
            
            elif cmd == 'delete':
                if len(args) >= 1:
                    notebook.delete(int(args[0]))
                else:
                    print("用法: delete <编号>")
            
            elif cmd == 'search':
                if len(args) >= 1:
                    notebook.search(args[0])
                else:
                    print("用法: search <关键词>")
            
            elif cmd == 'export':
                notebook.export()
            
            else:
                print(f"❓ 未知命令: '{cmd}'。输入 help 查看可用命令。")
        
        except KeyboardInterrupt:
            print("\n\n👋 收到中断信号，安全退出...")
            break
        except Exception as e:
            print(f"❌ 发生错误: {e}")


if __name__ == "__main__":
    main()

"""
个人密码管理器 PasswordManager v1.0
功能: CRUD / 主密码验证 / 强度评估 / 随机密码生成 / 备份
安全提示: 这是学习项目！生产环境请使用专业工具如 KeePass/Bitwarden
"""

import json
import os
import sys
import base64
import string
import random
import hashlib
import time
from datetime import datetime
from dataclasses import dataclass, asdict
from typing import List, Optional, Dict
from enum import Enum


# ========== 安全相关 ==========
def simple_encrypt(text: str, key: str = "default") -> str:
    """简单异或加密（仅用于学习演示！生产环境不可用！）"""
    key_bytes = key.encode()
    text_bytes = text.encode()
    encrypted = bytes(b ^ key_bytes[i % len(key_bytes)] for i, b in enumerate(text_bytes))
    return base64.b64encode(encrypted).decode()


def simple_decrypt(encoded: str, key: str = "default") -> str:
    """解密"""
    key_bytes = key.encode()
    encrypted = base64.b64decode(encoded.encode())
    decrypted = bytes(b ^ key_bytes[i % len(key_bytes)] for i, b in enumerate(encrypted))
    return decrypted.decode()


class StrengthLevel(Enum):
    WEAK = ("弱", "\033[31m")     # 红色
    MEDIUM = ("中", "\033[33m")   # 黄色
    STRONG = ("强", "\033[32m")   # 绿色
    RESET = "\033[0m"


def assess_strength(password: str) -> tuple:
    """
    评估密码强度
    返回: (等级枚举, 详细说明)
    """
    score = 0
    reasons = []
    
    length = len(password)
    if length >= 8:
        score += 1
    elif length < 6:
        reasons.append("太短了(至少8位)")
    else:
        reasons.append("建议至少8位")
    
    if any(c.islower() for c in password):
        score += 1
    else:
        reasons.append("缺少小写字母")
    
    if any(c.isupper() for c in password):
        score += 1
    else:
        reasons.append("缺少大写字母")
    
    if any(c.isdigit() for c in password):
        score += 1
    else:
        reasons.append("缺少数字")
    
    special_chars = set(string.punctuation)
    if any(c in special_chars for c in password):
        score += 1
    else:
        reasons.append("缺少特殊字符(!@#$等)")
    
    if len(set(password)) < len(password) * 0.6:
        reasons.append("重复字符太多")
        score -= 1
    
    # 评定等级
    if score <= 2:
        level = StrengthLevel.WEAK
    elif score <= 4:
        level =StrengthLevel.MEDIUM
    else:
        level = StrengthLevel.STRONG
    
    detail = f"得分:{score}/5 " + (", ".join(reasons) if reasons else "非常棒!")
    return level, detail


def generate_password(
    length: int = 16,
    use_upper: bool = True,
    use_digits: bool = True,
    use_symbols: bool = True,
) -> str:
    """生成随机密码"""
    chars = string.ascii_lowercase
    if use_upper:
        chars += string.ascii_uppercase
    if use_digits:
        chars += string.digits
    if use_symbols:
        chars += "!@#$%^&*()_+-=[]{}|;:,.<>?"
    
    # 确保每种类型至少有一个字符
    password_chars = [random.choice(string.ascii_lowercase)]
    if use_upper:
        password_chars.append(random.choice(string.ascii_uppercase))
    if use_digits:
        password_chars.append(random.choice(string.digits))
    if use_symbols:
        password_chars.append(random.choice("!@#$%^&*()_+-=[]{}|;:,.<>?"))
    
    # 填充剩余长度
    remaining = length - len(password_chars)
    password_chars.extend(random.choices(chars, k=remaining))
    
    # 打乱顺序
    random.shuffle(password_chars)
    return "".join(password_chars)

class PasswordManager:
    """密码管理器"""
    
    def __init__(self, data_file="passwords.json"):
        self.data_file = data_file
        self.master_key: Optional[str] = None
        self.entries: List[PasswordEntry] = []
        self.is_authenticated = False
        self._load()
    
    # ---------- 认证 ----------
    
    def set_master_password(self, password: str) -> bool:
        """设置/验证主密码"""
        config_file = ".pm_config"
        
        if os.path.exists(config_file):
            # 已有主密码，验证它
            with open(config_file, "r") as f:
                stored_hash = f.read().strip()
            
        input_hash = hashlib.sha256(password.encode()).hexdigest()
        
        if not os.path.exists(config_file):
            # 首次设置
            with open(config_file, "w") as f:
                f.write(input_hash)
            self.master_key = password
            self.is_authenticated = True
            return True
        
        if input_hash == stored_hash:
            self.master_key = password
            self.is_authenticated = True
            return True
        
        self.is_authenticated = False
        return False
    
    def require_auth(self) -> bool:
        """要求认证状态"""
        if not self.is_authenticated:
            print("❌ 需要先通过主密码认证!")
            return False
        return True
    
    # ---------- 数据持久化 ----------
    
    def _load(self):
        """加载数据"""
        if not os.path.exists(self.data_file):
            return
        try:
            with open(self.data_file, "r", encoding="utf-8") as f:
                data = json.load(f)
            self.entries = [
                PasswordEntry(**item) for item in data
            ]
        except Exception as e:
            print(f"⚠️ 加载数据失败: {e}")
    
    def _save(self):
        """保存数据"""
        data = [asdict(entry) for entry in self.entries]
        with open(self.data_file, "w", encoding="utf-8") as f:
            json.dump(data, f, ensure_ascii=False, indent=2)
    
    # ---------- CRUD ----------
    
    def add_entry(self, platform: str, username: str, password: str) -> bool:
        """添加密码条目"""
        if not self.require_auth():
            return False
        
        level, detail = assess_strength(password)
        enc_pwd = simple_encrypt(password, self.master_key)
        
        entry = PasswordEntry(
            platform=platform,
            username=username,
            encrypted_password=enc_pwd,
        )
        self.entries.append(entry)
        self._save()
        
        color = level.value[1]
        reset = StrengthLevel.RESET.value
        strength_name = level.value[0]
        
        print(f"✅ 已保存: {platform} / {username}")
        print(f"   强度: {color}{strength_name}{reset} ({detail})")
        return True
    
    def list_entries(self):
        """列出所有账户"""
        if not self.entries:
            print("📭 还没有保存任何密码")
            return
        
        print(f"\n{'='*55}")
        print(f"📋 共 {len(self.entries)} 个密码条目")
        print('='*55)
        print(f"  {'序号':<4} {'平台':<18} {'用户名':<22} {'日期'}")
        print(f"  {'-'*52}")
        
        for i, entry in enumerate(self.entries, 1):
            date_short = entry.created_at[:10]
            print(f"  {i:<4} {entry.platform:<18} {entry.username:<22} {date_short}")
    
    def get_password(self, platform: str) -> bool:
        """获取指定平台的密码（需重新验证主密码）"""
        if not self.require_auth():
            return False
        
        for entry in self.entries:
            if entry.platform.lower() == platform.lower():
                # 再次验证主密码
                verify = input("请输入主密码确认: ")
                if hashlib.sha256(verify.encode()).hexdigest() != \
                   hashlib.sha256(self.master_key.encode()).hexdigest():
                    print("❌ 主密码错误!")
                    return False
                
                decrypted = simple_decrypt(entry.encrypted_password, self.master_key)
                level, _ = assess_strength(decrypted)
                
                print(f"\n  🔓 {entry.platform}")
                print(f"  用户名: {entry.username}")
                print(f"  密码:   {decrypted}")
                print(f"  强度:   {level.value[0]}")
                
                # 自动复制到剪贴板提示（可选）
                input("\n  按Enter键清除屏幕上的密码...")
                return True
        
        print(f"❌ 未找到平台: {platform}")
        return False
    
    def delete_entry(self, platform: str) -> bool:
        if not self.require_auth():
            return False
        
        original_len = len(self.entries)
        self.entries = [
            e for e in self.entries
            if e.platform.lower() != platform.lower()
        ]
        
        if len(self.entries) < original_len:
            self._save()
            print(f"🗑 已删除: {platform}")
            return True
        print(f"❌ 未找到: {platform}")
        return False
    
    def search(self, keyword: str):
        """搜索"""
        results = [
            e for e in self.entries
            if keyword.lower() in e.platform.lower()
            or keyword.lower() in e.username.lower()
        ]
        
        if not results:
            print(f"🔍 未找到匹配: '{keyword}'")
            return
        
        print(f"\n🔍 找到 {len(results)} 条结果:")
        for r in results:
            print(f"  • {r.platform} / {r.username}")
    
    def backup(self) -> str:
        """备份数据"""
        timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
        backup_file = f"passwords_backup_{timestamp}.json"
        
        import shutil
        shutil.copy2(self.data_file, backup_file)
        print(f"💾 备份完成: {backup_file}")
        return backup_file
