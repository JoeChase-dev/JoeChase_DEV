## 三、LeetCode进阶题（每题5分，共30分）

### 🏆 第16题 — LC #121 买卖股票的最佳时机 ⭐⭐
给定一个数组 `prices`，其中 `prices[i]` 是股票第 `i` 天的价格。你只能选择**某一天**买入并在**未来的某一个不同的日子**卖出。设计一个算法来计算你所能获取的最大利润。  
示例：`prices = [7,1,5,3,6,4]` → 输出：`5`（第2天买入，第5天卖出）

---

### 🏆 第17题 — LC #53 最大子数组和 ⭐⭐
给你一个整数数组 `nums`，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。  
示例：`nums = [-2,1,-3,4,-1,2,1,-5,4]` → 输出：`6`（子数组 `[4,-1,2,1]`）

---

### 🏆 第18题 — LC #283 移动零 ⭐⭐
给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。  
示例：`nums = [0,1,0,3,12]` → 输出：`[1,3,12,0,0]`

---

### 🏆 第19题 — LC #442 数组中重复的数据 ⭐⭐
给定一个整数数组 `nums`，其中 `1 ≤ a[i] ≤ n`（n为数组长度），其中有些元素出现了两次而其他元素出现了一次。找到所有出现了两次的元素。  
示例：`nums = [4,3,2,7,8,2,3,1]` → 输出：`[2,3]`

---

### 🏆 第20题 — LC #98 验证二叉搜索树 ⭐⭐⭐
给你一个二叉树的根节点 `root`，判断其是否是一个有效的二叉搜索树（BST）。  
BST定义：左子树所有节点 < 根节点 < 右子树所有节点。
