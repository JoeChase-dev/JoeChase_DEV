### 🔘 第2题 — Git合并 ⭐⭐
`git merge` 和 `git rebase` 的区别是什么？

**正确答案: B** ✅ → **`merge` 创建新的合并提交，`rebase` 重写提交历史**

```bash
# merge：保留完整历史，创建合并提交
# 图形：
# *   merge commit
# |\  
# | * feature commit
# * | main commit
# |/
# * common commit

# rebase：线性历史，不创建合并提交
# 图形：
# * feature commit (重写后的)
# * main commit
# * common commit
```

| 对比 | merge | rebase |
|------|-------|--------|
| 历史 | 保留分支历史（非线性） | 线性历史（更整洁） |
| 提交 | 创建新的合并提交 | 重写提交（改变commit ID） |
| 安全性 | ✅ 安全（不修改历史） | ⚠️ 危险（不要对公共分支rebase） |
| 使用场景 | 合并公共分支 | 整理本地提交（未push前） |

### 🔘 第4题 — Git撤销操作 ⭐⭐

**正确答案: B** ✅ → **`git commit --amend`**

| 场景 | 命令 | 说明 |
|------|------|------|

### 💻 第13题 — Git Rebase操作 ⭐⭐⭐

**作用**：`git rebase main` 将当前分支的提交"移植"到 main 分支的最新提交之后，形成线性历史。

**操作步骤**：
```bash
# 场景：在 feature 分支开发，main 分支有新的提交

# ① 确保 main 分支是最新的
git checkout main
git pull origin main

# ② 切换到 feature 分支
git checkout feature

# ③ 执行 rebase
git rebase main

# 如果出现冲突：
# - 手动解决冲突
# - git add <conflict-file>
# - git rebase --continue
# - 或 git rebase --abort（放弃rebase）

# ④ 推送（需要强制推送，因为历史已改写）
git push origin feature --force-with-lease
# --force-with-lease 比 --force 更安全（防止覆盖他人提交）
```

**rebase vs merge 图形对比**：
```
# merge（保留分支历史）：
*   merge commit
|\  
| * feature commit 2
* | main commit 2
| * feature commit 1
* | main commit 1
|/
* common commit

# rebase（线性历史）：
* feature commit 2 (重写后)
* feature commit 1 (重写后)
* main commit 2
* main commit 1
* common commit
```

### 🏆 第16题 — LC #121 买卖股票的最佳时机 ⭐⭐

**题目**：只能选择某一天买入并在未来的某一个不同的日子卖出，求最大利润。

**解法：一次遍历 — O(n)**
```python
def max_profit(prices):
    if not prices:
        return 0
    
    min_price = float('inf')
    max_profit = 0
    
    for price in prices:
        min_price = min(min_price, price)          # 更新最低买入价
        profit = price - min_price                  # 计算当前卖出利润
        max_profit = max(max_profit, profit)        # 更新最大利润
    
    return max_profit

# 测试：
print(max_profit([7,1,5,3,6,4]))   # 5（第2天买入1，第5天卖出6）
print(max_profit([7,6,4,3,1]))      # 0（价格一直跌，不买卖）
```

**思路**：遍历过程中维护两个变量：
- `min_price`：到当前位置的最低价格
- `max_profit`：到当前位置的最大利润

### 🏆 第17题 — LC #53 最大子数组和 ⭐⭐

**题目**：找出具有最大和的连续子数组。

**解法1：Kadane算法 — O(n)**
```python
def max_sub_array(nums):
    max_sum = nums[0]
    current_sum = 0
    
    for num in nums:
        current_sum = max(num, current_sum + num)  # 关键！
        max_sum = max(max_sum, current_sum)
    
    return max_sum

# 测试：
print(max_sub_array([-2,1,-3,4,-1,2,1,-5,4]))   # 6（[4,-1,2,1]）
print(max_sub_array([1]))                          # 1
print(max_sub_array([5,4,-1,7,8]))               # 23（整个数组）
```

**思路**：
- `current_sum`：以当前元素结尾的最大子数组和
- 要么把当前元素加入之前的子数组，要么从当前元素重新开始
- `current_sum = max(num, current_sum + num)`

**解法2：动态规划**
```python
def max_sub_array_dp(nums):
    dp = [0] * len(nums)
    dp[0] = nums[0]
    max_sum = dp[0]
    
    for i in range(1, len(nums)):
        dp[i] = max(nums[i], dp[i-1] + nums[i])
        max_sum = max(max_sum, dp[i])
    
    return max_sum
```

### 🏆 第18题 — LC #283 移动零 ⭐⭐

**题目**：将所有0移动到数组末尾，保持非零元素相对顺序。

**解法1：双指针 — O(n)**
```python
def move_zeroes(nums):
    # 方法：把非零元素移到前面，后面补0
    left = 0  # 指向下一个可以放置非零元素的位置
    
    # 第一遍：把非零元素移到前面
    for right in range(len(nums)):
        if nums[right] != 0:
            nums[left] = nums[right]
            left += 1
    
    # 第二遍：后面补0
    for i in range(left, len(nums)):
        nums[i] = 0

# 测试：
nums = [0,1,0,3,12]
move_zeroes(nums)
print(nums)   # [1, 3, 12, 0, 0]
```

**解法2：更简洁的交换版本**
```python
def move_zeroes_swap(nums):
    left = 0
    for right in range(len(nums)):
        if nums[right] != 0:
            nums[left], nums[right] = nums[right], nums[left]
            left += 1

# 测试：
nums = [0,1,0,3,12]
move_zeroes_swap(nums)
print(nums)   # [1, 3, 12, 0, 0]
```

### 🏆 第19题 — LC #442 数组中重复的数据 ⭐⭐⭐

**题目**：给定数组，1 ≤ a[i] ≤ n，找出所有出现两次的元素。

**解法1：取负标记法（O(1)空间）— O(n)**
```python
def find_duplicates(nums):
    result = []
    
    for num in nums:
        index = abs(num) - 1  # 因为 1 ≤ a[i] ≤ n，所以减1作为索引
        if nums[index] < 0:
            result.append(abs(num))  # 已经是负数，说明之前出现过
        else:
            nums[index] *= -1  # 标记为已出现
    
    return result

# 测试：
print(find_duplicates([4,3,2,7,8,2,3,1]))   # [2, 3]
print(find_duplicates([1,1,2]))                # [1]
```

**思路**：
- 遍历数组，对于每个元素 `num`，把 `nums[abs(num)-1]` 取负
- 如果 `nums[abs(num)-1]` 已经是负数，说明 `num` 之前出现过（重复）

**解法2：使用集合 — O(n)空间**
```python
def find_duplicates_set(nums):
    seen = set()
    result = []
    
    for num in nums:
        if num in seen:
            result.append(num)
        else:
            seen.add(num)
    
    return result
```

### 🏆 第20题 — LC #98 验证二叉搜索树 ⭐⭐⭐

**题目**：判断给定的二叉树是否是有效的二叉搜索树（BST）。

**解法1：中序遍历（BST中序遍历是升序）— O(n)**
```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def is_valid_bst_inorder(root):
    def inorder(node):
        if not node:
            return []
        return inorder(node.left) + [node.val] + inorder(node.right)
    
    inorder_list = inorder(root)
    return inorder_list == sorted(inorder_list) and len(inorder_list) == len(set(inorder_list))
```

**解法2：递归（传递上下界）— O(n)**
```python
def is_valid_bst(root, min_val=float('-inf'), max_val=float('inf')):
    if not root:
        return True
    
    # 当前节点必须满足：min_val < root.val < max_val
    if root.val <= min_val or root.val >= max_val:
        return False
    
    # 左子树的所有节点 < root.val
    # 右子树的所有节点 > root.val
    return (is_valid_bst(root.left, min_val, root.val) and
            is_valid_bst(root.right, root.val, max_val))

# 测试：
#     2
#    / \
#   1   3
root = TreeNode(2, TreeNode(1), TreeNode(3))
print(is_valid_bst(root))   # True

#     5
#    / \
#   1   4
#      / \
#     3   6
root = TreeNode(5, TreeNode(1), TreeNode(4, TreeNode(3), TreeNode(6)))
print(is_valid_bst(root))   # False（右子树的3 < 5）
```
