- `push(item)`：入栈
- `pop()`：出栈（返回栈顶元素）
- `peek()`：查看栈顶元素（不出栈）
- `is_empty()`：判断栈是否为空

### ✍️ 第12题 — 用Python实现队列 ⭐⭐

- `is_empty()`：判断队列是否为空
