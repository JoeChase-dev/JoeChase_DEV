### ✍️ 第11题 — 用Python实现栈 ⭐⭐

**参考实现**：
```python
class Stack:
    def __init__(self):
        self.items = []
    
    def push(self, item):
        """入栈"""
        self.items.append(item)
    
    def pop(self):
        """出栈（返回栈顶元素）"""
        if self.is_empty():
            raise IndexError("Stack is empty")
        return self.items.pop()
    
    def peek(self):
        """查看栈顶元素（不出栈）"""
        if self.is_empty():
            raise IndexError("Stack is empty")
        return self.items[-1]
    
    def is_empty(self):
        """判断栈是否为空"""
        return len(self.items) == 0
    
    def size(self):
        """返回栈的大小"""
        return len(self.items)
    
    def __str__(self):
        return str(self.items)

# 测试：
s = Stack()
s.push(1)
s.push(2)
s.push(3)
print(s)           # [1, 2, 3]
print(s.pop())     # 3
print(s.peek())    # 2
print(s.is_empty()) # False
```

**得分点**：
- ✅ 使用 `list` 作为底层存储（2分）
- ✅ `push` 用 `append()`（1分）
- ✅ `pop` 用 `pop()`（1分）
- ✅ 错误处理（栈空时）（2分）
- ✅ `peek` 实现正确（2分）

### ✍️ 第12题 — 用Python实现队列 ⭐⭐

**参考实现**：
```python
from collections import deque

class Queue:
    def __init__(self):
        self.items = deque()
    
    def enqueue(self, item):
        """入队（从右侧加入）"""
        self.items.append(item)
    
    def dequeue(self):
        """出队（从左侧弹出）"""
        if self.is_empty():
            raise IndexError("Queue is empty")
        return self.items.popleft()
    
    def front(self):
        """查看队首元素"""
        if self.is_empty():
            raise IndexError("Queue is empty")
        return self.items[0]
    
    def is_empty(self):
        """判断队列是否为空"""
        return len(self.items) == 0
    
    def size(self):
        """返回队列大小"""
        return len(self.items)

# 测试：
q = Queue()
q.enqueue(1)
q.enqueue(2)
q.enqueue(3)
print(q.dequeue())  # 1
print(q.front())    # 2
print(q.is_empty()) # False
```

> 💡 **为什么用 `deque` 而不是 `list`？**
> - `list.pop(0)` 是 O(n)（需要移动所有元素）
> - `deque.popleft()` 是 O(1)（双端队列的优势）

### ✍️ 第13题 — 手写链表节点和遍历 ⭐⭐

**参考实现**：
```python
class ListNode:
    """链表节点"""
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def print_list(head):
    """遍历并打印链表"""
    curr = head
    while curr:
        print(curr.val, end=" -> " if curr.next else "\n")
        curr = curr.next

def create_list(vals):
    """根据数组创建链表"""
    if not vals:
        return None
    head = ListNode(vals[0])
    curr = head
    for val in vals[1:]:
        curr.next = ListNode(val)
        curr = curr.next
    return head

# 测试：
head = create_list([1, 2, 3, 4, 5])
print_list(head)   # 1 -> 2 -> 3 -> 4 -> 5
```

**进阶：反转链表（LeetCode #206）**：
```python
def reverse_list(head):
    """反转链表（迭代法）"""
    prev = None
    curr = head
    while curr:
        next_temp = curr.next   # 暂存下一个节点
        curr.next = prev        # 反转指针
        prev = curr             # prev 后移
        curr = next_temp        # curr 后移
    return prev

# 测试：
head = create_list([1, 2, 3])
reversed_head = reverse_list(head)
print_list(reversed_head)   # 3 -> 2 -> 1
```

### 🏆 第15题 — LC #1 两数之和 ⭐

**题目**：给定整数数组 `nums` 和目标值 `target`，找出和为目标值的两个整数的数组下标。

**解法1：哈希表（推荐）— O(n)**
```python
def two_sum(nums, target):
    """哈希表解法"""
    seen = {}                       # 值 → 索引 的映射
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
    
    return []                       # 没找到

# 测试：
print(two_sum([2, 7, 11, 15], 9))   # [0, 1]
print(two_sum([3, 2, 4], 6))        # [1, 2]
```

**解法2：暴力法 — O(n²)**
```python
def two_sum_bruteforce(nums, target):
    for i in range(len(nums)):
        for j in range(i + 1, len(nums)):
            if nums[i] + nums[j] == target:
                return [i, j]
    return []
```

### 🏆 第16题 — LC #20 有效的括号 ⭐

**题目**：判断字符串中的括号是否有效匹配。

**解法：栈 — O(n)**
```python
def is_valid(s):
    """栈解法"""
    stack = []
    mapping = {')': '(', ']': '[', '}': '{'}
    
    for char in s:
        if char in mapping:              # 是右括号
            top = stack.pop() if stack else '#'  # 弹出栈顶
            if mapping[char] != top:
                return False
        else:                            # 是左括号
            stack.append(char)
    
    return not stack                     # 栈为空才有效

# 测试：
print(is_valid("()"))           # True
print(is_valid("()[]{}"))      # True
print(is_valid("(]"))          # False
print(is_valid("([)]"))        # False
print(is_valid("{[]}"))        # True
```

### 🏆 第17题 — LC #21 合并两个有序链表 ⭐⭐

**题目**：合并两个升序链表为一个新的升序链表。

**解法1：迭代 — O(n+m)**
```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def merge_two_lists(l1, l2):
    """迭代法"""
    dummy = ListNode(0)          # 哑节点（简化代码）
    curr = dummy
    
    while l1 and l2:
        if l1.val <= l2.val:
        curr.next = l1
        l1 = l1.next
    else:
        curr.next = l2
        l2 = l2.next
    curr = curr.next
    
    # 处理剩余节点
    curr.next = l1 if l1 else l2
    
    return dummy.next

# 解法2：递归（更简洁）
def merge_two_lists_recursive(l1, l2):
    if not l1:
        return l2
    if not l2:
        return l1
    
    if l1.val <= l2.val:
        l1.next = merge_two_lists_recursive(l1.next, l2)
        return l1
    else:
        l2.next = merge_two_lists_recursive(l1, l2.next)
        return l2
```

### 🏆 第18题 — LC #70 爬楼梯 ⭐

**题目**：每次可以爬1或2个台阶，爬n阶台阶有多少种方法？

**解法：动态规划 — O(n)**
```python
def climb_stairs(n):
    """动态规划"""
    if n <= 2:
        return n
    
    # dp[i] = dp[i-1] + dp[i-2]
    prev2, prev1 = 1, 1          # dp[0], dp[1]
    for i in range(2, n + 1):
        curr = prev1 + prev2
        prev2 = prev1
        prev1 = curr
    
    return prev1

# 测试：
print(climb_stairs(2))   # 2（1+1, 2）
print(climb_stairs(3))   # 3（1+1+1, 1+2, 2+1）
print(climb_stairs(4))   # 5

# 数学原理：斐波那契数列！
# ways(n) = ways(n-1) + ways(n-2)
```

### 🏆 第20题 — LC #704 二分查找 ⭐

**题目**：在有序数组中查找目标值，返回下标或-1。

**解法：二分查找 — O(log n)**
```python
def search(nums, target):
    """二分查找（迭代法）"""
    left, right = 0, len(nums) - 1
    
    while left <= right:
        mid = (left + right) // 2
        
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return -1

# 测试：
print(search([-1,0,3,5,9,12], 9))   # 4
print(search([-1,0,3,5,9,12], 2))   # -1
```

**参考答案**：

| 对比维度 | BFS（广度优先搜索） | DFS（深度优先搜索） |
|---------|-------------------|-------------------|
| 数据结构 | 队列（Queue） | 栈（Stack）或递归 |
| 遍历顺序 | 层序（先访问邻接节点） | 纵深（一条路走到黑） |
| 空间复杂度 | O(w) w为最大宽度 | O(h) h为最大深度 |
| 最短路径 | ✅ 适合（无权图） | ❌ 不适合 |
| 连通性 | ✅ 可以 | ✅ 可以 |
| 记忆化 | 不太需要 | ✅ 经常需要（DP问题） |

**应用场景**：
- **BFS**：最短路径（无权图）、社交网络推荐（六度分隔）、层序遍历二叉树
- **DFS**：拓扑排序、连通分量、迷宫求解、回溯法（八皇后问题）

### 📝 第22题（6分）— 哈希表冲突解决 ⭐⭐

**参考答案**：

| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **链地址法（Chaining）** | 每个桶存一个链表，冲突的元素追加到链表 | 实现简单，删除方便 | 链表过长时退化为O(n) |
| **开放寻址法（Open Addressing）** | 冲突时，按某种探测序列找下一个空桶 | 不需要额外内存存储指针 | 删除复杂，聚集问题 |
| **双重哈希（Double Hashing）** | 使用两个哈希函数，冲突时用第二个哈希计算结果 | 分布更均匀 | 计算开销更大 |
| **再哈希（Rehashing）** | 负载因子过高时，扩大哈希表并重新哈希所有元素 | 保持操作效率 | 扩容时有一定开销 |

**Python dict 的实现**：
- 使用**开放寻址法**（具体是"伪随机探测"）
- 负载因子 > 2/3 时触发扩容
- 为什么不用链地址法？— Python追求内存效率，开放寻址缓存友好性更好
