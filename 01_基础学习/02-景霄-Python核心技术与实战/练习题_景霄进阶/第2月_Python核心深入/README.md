## ✅ 使用建议

### 学习流程
```
看完教程 → 做对应周题目版(自测) → 对比详解版查漏补缺 → 记录错题 → LeetCode实战 → 复习巩固
```

### 使用技巧
1. **先做题再看答案** — 不要直接看详解版，否则效果大打折扣
2. **标记错题** — 用 `❌` 标记做错的题，考前重点复习
3. **计时完成** — 模拟考试环境，训练解题速度
4. **动手验证** — 不确定的代码一定要跑一遍，不要靠猜
5. **定期重刷** — 每周末把本周的错题再做一遍
6. **LeetCode同步** — 学完每周理论后，立即完成对应的LC题目

### 难度标注
- ⭐ 基础题 — 必须全部掌握
- ⭐⭐ 进阶题 — 大部分要会做
- ⭐⭐⭐ 挑战题 — 锻炼思维，面试常考
