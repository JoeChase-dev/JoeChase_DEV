## ⚡ GitHub 使用技巧

### 快速找项目
```
site:github.com [关键词] stars:>1000
```
例如：`site:github.com RAG stars:>5000`

### 发现好东西
- `trending` 页面：看今天/本周热门项目
- `awesome-xxx` 系列：`awesome-python` `awesome-vue` `awesome-docker`
- `topics` 标签系统：搜索 `topic:rag` `topic:fastapi`

### 参与开源（从 good first issue 开始）
```
label:good first issue language:python
```
