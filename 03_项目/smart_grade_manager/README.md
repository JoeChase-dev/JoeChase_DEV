# 🎓 智能学生成绩管理系统 (Smart Grade Manager)

> **基于V5学习规划第1个月知识点的完整实战项目**

一个命令行交互式的学生成绩管理系统，涵盖Python基础语法、面向对象、高级特性、文件操作、正则表达式、装饰器等核心知识点。

---

## 📚 本项目覆盖的知识点

### 🔹 Week 1: 基础语法
| 知识点 | 项目中的应用 |
|--------|-------------|
| 变量与数据类型 | 学生信息的存储（int/float/str/bool） |
| 运算符 | 成绩计算（平均分、总分、排名） |
| 分支结构 (if/elif/else) | 成绩等级判定、菜单选项处理 |
| 循环结构 (for/while) | 遍历学生列表、批量操作 |
| 函数定义与调用 | 各功能模块的封装 |
| 字符串操作 | 学号格式化、姓名搜索、信息展示 |
| 列表/字典/集合 | 学生数据容器、快速查找 |

### 🔹 Week 2: 面向对象
| 知识点 | 项目中的应用 |
|--------|-------------|
| 类与对象 | `Student`类、`GradeManager`类 |
| 继承 | 自定义异常继承体系 |
| 多态 | 不同类型学生的差异化处理 |
| 魔术方法 | `__str__`、`__repr__`、`__lt__`、`__eq__` |
| 异常处理 | 数据验证、文件操作的错误捕获 |
| 文件I/O | JSON格式的数据持久化 |
| 模块与包 | 项目的分层架构设计 |

### 🔹 Week 3: 高级特性
| 知识点 | 项目中的应用 |
|--------|-------------|
| 装饰器 | 日志记录、性能计时、输入验证、权限控制 |
| 生成器 | 大数据量的分批加载、成绩排名流式生成 |
| 迭代器 | 自定义学生集合的遍历 |
| 列表推导式 | 数据筛选和转换的一行代码 |
| Lambda函数 | 排序key、过滤条件、映射转换 |
| 正则表达式 | 学号/手机号/邮箱格式验证 |
| 多线程基础 | 异步保存数据（不阻塞主界面） |

### 🔹 Week 4: 综合实战 + Git

## 📁 项目结构

```
smart_grade_manager/
├── README.md                    # 本文件
├── requirements.txt             # 依赖列表（纯标准库，无需安装）
├── main.py                      # 🎯 主程序入口
├── models/
│   ├── __init__.py
│   └── student.py               # 学生数据模型（OOP核心）
│   ├── exceptions.py            # 自定义异常类
│   └── enums.py                 # 枚举定义
├── utils/
│   ├── __init__.py
│   ├── decorators.py            # ⭐ 装饰器工具集
│   ├── validators.py            # ✅ 输入验证（正则）
│   ├── file_io.py               # 📁 文件读写
│   └── formatters.py            # 🎨 格式化输出
├── services/
│   ├── __init__.py
│   └── grade_service.py         # 核心业务逻辑
├── tests/
│   ├── __init__.py
│   └── test_system.py           # 单元测试
└── data/
    └── students.json            # 数据存储文件（运行后自动创建）
```

## 🚀 快速开始

```bash
# 1. 进入项目目录
cd smart_grade_manager

# 2. 运行程序（纯Python标准库，无需pip install）
python main.py

# 3. 按照菜单提示操作即可
```
